The `Rating` element renders one to five stars depending on what the given rating is.

## Props
```ts
interface Props {
  rating?: number | null
}

const { rating } = Astro.props
```
| Prop | Default | Purpose |
| ----------- | ----------- | ----------- |
| rating | None | The amount of stars shown (from 1 to 5) |

## Example code

```js title="index.astro"
<Rating rating={2} />
```
<Window>
  <Rating rating={2} />
</Window>

```js title="index.astro"
<Rating rating={5} />
```

<Window>
