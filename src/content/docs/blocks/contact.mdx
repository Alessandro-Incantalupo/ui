The `Contact` component is used to render your contact information, typically on a separate page. 

## Props

```ts
interface Props
  extends Pick<
    ComponentProps<typeof Section>,
    'level' | 'tagline' | 'heading' | 'text' | 'image'
  > {}

const { props } = Astro
```

| Prop | Default | Purpose |
| ----------- | ----------- | ----------- |
| level | None | Text shown on the button |
| tagline | primary | Style of the button |
| heading | None | Link to where the button should bring you |
| text | None | Icon displayed on the button |
| image | None | Image |

```js
<ContactSection text="Bla" heading="Contact" />
```
