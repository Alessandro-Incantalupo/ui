The `Social` element is used to render an icon of a social media platform that links directly to your desired destination.

## Props

```ts
export interface Props {
  key: 'facebook' 
    | 'instagram' 
    | 'twitter' 
    | 'pinterest' 
    | 'linkedin' 
    | 'youtube' 
    | 'tiktok' 
    | 'snapchat'
  value: string
}
```

More platforms may be added in the future.

| Prop | Default | Purpose |
| ----------- | ----------- | ----------- |
| key | None | What social you want to display |
| value | None | Your username on that social/platform |


## Example code

```js
<Social key="instagram" value="example" />
```
