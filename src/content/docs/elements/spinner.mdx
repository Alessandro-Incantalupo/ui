The `Spinner` component makes any icon that you pass through spin. This is usually used to show something is loading.

```ts
export interface Props {
  icon?: string
}
```

| Prop | Default | Purpose |
| ----------- | ----------- | ----------- |
| icon | loader-2 | The icon you want to be shown in the spinner |

## Example code

```js
<Spinner icon="loader-2" />
```
