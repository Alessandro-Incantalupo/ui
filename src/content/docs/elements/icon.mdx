The `Icon` element is used, as the name implies, to render a single icon.

## Props
```ts
interface Props {
  icon?: keyof typeof tabler.icons
  large?: boolean
  class?: string
}
```

| Prop | Default | Purpose |
| ----------- | ----------- | ----------- |
| icon | None | The icon you want to dispaly from the Tabler Icon Pack |
| large | false | Whether to make the icon large |
| class | None | Any styling classes you want to add to the icon |

## Example code

```js title="index.astro"
<Icon icon="activity-heartbeat" />
```
