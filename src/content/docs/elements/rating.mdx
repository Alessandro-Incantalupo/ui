The `Rating` element renders one to five stars depending on what the given rating is.

## Props
```ts
export interface Props {
  rating?: number | null
}
```
| Prop | Default | Purpose |
| ----------- | ----------- | ----------- |
| rating | None | The amount of stars shown (from 1 to 5) |

## Example code

```js
<Rating rating={2} />
```
