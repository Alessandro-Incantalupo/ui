The `Textarea` element is used to display long amount of text, usually inside of a form.

## Props
```ts
interface Props {
  label?: string
  required?: boolean
}

const { label, required } = Astro.props
```

| Prop | Default | Purpose |
| ----------- | ----------- | ----------- |
| label | None | Text shown above the textarea |
| required | None | Whether the textarea is required to continue |

## Example code

```astro title="index.astro"
<Textarea>
  Text
</Textarea>
```
