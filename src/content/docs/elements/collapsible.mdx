The `Collapsible` is used to make a collapsible that shows content upon clicking on the text.

## Props
```ts
interface Props {
  title: string
  text?: string
}

const { title, text } = Astro.props
```

| Prop | Default | Purpose |
| ----------- | ----------- | ----------- |
| title | None | Shown as text that a user must click on to display the content |
| text | None | The content shown when the collapsible is open |

## Example code
```js title="index.astro"
<Collapsible title="Example collapsible, click me!">
  This content is shown while the collapsible is open, then hidden again when closed.
</Collapsible>
```
