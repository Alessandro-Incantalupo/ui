The `Button` element is self explanatory, it is a button that receives props from where the element is being called.
## Props
```ts
export interface Props extends ComponentProps<typeof Icon> {
  text?: string | null
  area?: 'primary' | 'secondary'
  href?: HTMLAttributes<'a'>['href'] | null
  icon?: string
}
```

| Prop | Default | Purpose |
| ----------- | ----------- | ----------- |
| text | None | Text shown on the button |
| area | primary | Style of the button |
| href | None | Link to where the button should bring you |
| icon | None | Icon displayed on the button |

## Example code
```js title="index.astro"
<Button href="#" icon="arrow-right">Test</Button>
```
