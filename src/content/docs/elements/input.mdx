The `Input` element is used to render a single input. This element is usually used inside of a [Form](/elements/form).

## Props
```ts
export interface Props {
  label: string
  type?: 'text' | 'email' | 'tel' | 'url' | 'number'
  required?: boolean
}
```

| Prop | Default | Purpose |
| ----------- | ----------- | ----------- |
| label | None | The text shown above the input field |
| type | text | The type of the input field (HTML input types) |
| required | false | Whether the input field is required to continue |

## Example code

```js title="index.astro"
  <Input
    label="Email"
    type="email"
  />
```
