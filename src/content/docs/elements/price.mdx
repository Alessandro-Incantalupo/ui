The `Price` element turns any number into a two decimal number and puts the euro symbol infront of the number.

## Props

```ts
interface Props {
  price?: number | null
}

const { price } = Astro.props
```

| Prop | Default | Purpose |
| ----------- | ----------- | ----------- |
| price | None | The price of your item |

## Example code

```js title="index.astro"
<Price price={30} />
```
<Window>
  <Price price={30} />
</Window>

```js title="index.astro"
<Price price={29.9} />
```

<Window>
  <Price price={29.9} />
</Window>
