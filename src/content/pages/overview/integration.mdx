After <Link href="/installation">installation</Link> you can generate your color palettes via an Astro integration. You can optionally add a custom css file and define custom colors and other theming options, see the <Link href="/theming">theming page</Link> for more information.

```ts
import { defineConfig } from 'astro/config'
import fulldevUI from 'fulldev-ui/integration'

// https://astro.build/config
export default defineConfig({
  integrations: [
    fulldevUI({
      css: '/src/css/custom.css',
      colors: {
        theme: 'dark',
        dark: {
          background: '#111110',
          base: '#6F6D66',
          brand: '#F50',
        },
      },
    }),
  ],
})
```
