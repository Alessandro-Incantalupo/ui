## Installation & Usage

Installation is simple, after the <Link href="/installation">installation process</Link> just import the integration from the package and add it to the integrations array in your `astro.config.ts` file.

```ts props={{ title: 'astro.config.ts' }}
import { defineConfig } from 'astro/config'
import fullUI from 'fulldev-ui/integration'

// https://astro.build/config
export default defineConfig({
  site: 'https://example.com',
  integrations: [
    fullUI({
      css: '/src/css/custom.css',
      colors: {
        theme: 'dark',
        dark: {
          background: '#111110',
          base: '#6F6D66',
          brand: '#F50',
        },
      },
    }),
  ],
})
```
