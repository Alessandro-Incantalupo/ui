## Installation

Installing FullUI is simple, just install `fulldev-ui` with your favorite package manager and you are ready to go. To get started with the integration see the <Link href="/integration">integration page</Link>.

## Usage

After installing FullUI, you can start using it in your project by simply importing the components and blocks you need, on our docs you can find examples for all components and blocks.
