---
title: Child
---

Renders a child components and/or element if needed. For a more detailed explenation, see [this page](/overview/children).

## Example
``` astro live
---
import { Child } from '@fullui/ui'
---

<Child
  component="Heading"
  html="This is a heading"
/>

<Child
  component="Heading"
  as="p"
  html="A heading as a paragraph"
/>

<Child
  component="Heading"
  html="A heading "
  orEvenNested={{
    as: 'span',
    style: 'color:red;',
    html: 'with nested'
  }}
/>
```
