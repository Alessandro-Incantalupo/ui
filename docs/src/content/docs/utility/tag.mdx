This component is used in every component in Fullui. You probably don't need to use it directly, but it's here when you need it.

### Props
Extends [Polymorphic](https://docs.astro.build/en/guides/typescript/#polymorphic-type)
```ts
if?: any
color?: string
scheme?: string
contrast?: boolean
compact?: boolean
size?: '1' | '2' | '3' | '4' | 1 | 2 | 3 | 4
```

### Defaults
```ts
as = 'div'
```

## Example
```astro live
---
import { Tag } from '@fullui/ui'
---

<Tag> I'm rendered because my slot contains content </Tag>
<Tag></Tag> <!-- not rendered -->
```
