This component is used in every component in Fullui. Most of the time you won't need it, but you can use it to build your own components.

### Props
This component used [Polymorphic](https://docs.astro.build/en/guides/typescript/#polymorphic-type); a native Astro feature.

```ts
if?: any
color?: string
scheme?: string
contrast?: boolean
size?: 'small' | 'medium' | 'large'
compact?: boolean
```
For a more detailed exlenation on the props, please see [the Theming page](/overview/theming).

### Defaults
```ts
as = 'div'
```

## Example
```astro live
---
import { Tag } from '@fullui/ui'
---

<Tag> I'm rendered because my slot contains content </Tag>

<Tag></Tag> <!-- not rendered -->
```
