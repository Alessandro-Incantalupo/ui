## Props
Below props are available in addition to all [Tag](/foundation/tag) props.

#### `name`
**Type:** `name?: [TABLER ICON NAME]`

For all available icons see https://tablericons.com/.

```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon name="circle" />
<Icon name="arrow-right" />
<Icon name="brand-x" />
<Icon name="brand-github" />
```

#### `size`
**Type:** `size?: 'sm' | 'md' | 'lg'`

```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon size="sm" name="circle" />
<Icon size="md" name="circle" />
<Icon size="lg" name="circle" />
```

#### `compact`
**Type:** `compact?: boolean`

```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon compact size="sm" name="circle" />
<Icon compact size="md" name="circle" />
<Icon compact size="lg" name="circle" />
```
