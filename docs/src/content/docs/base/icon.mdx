An icon or a component that looks like a icon.

<br/>

```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon name="circle" />
```

## Props
### `name`
**Type:** `name?: [TABLER ICON NAME]`

For all available icons see https://tablericons.com/.

```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon name="circle" />
<Icon name="arrow-right" />
<Icon name="brand-x" />
<Icon name="brand-github" />
```

### `size`
**Type:** `size?: 'sm' | 'md' | 'lg'`

```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon size="sm" name="circle" />
<Icon size="md" name="circle" />
<Icon size="lg" name="circle" />
```

### `compact`
**Type:** `compact?: boolean`

```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon density="compact" size="sm" name="circle" />
<Icon density="compact" size="md" name="circle" />
<Icon density="compact" size="lg" name="circle" />
```

### `density`
**Type:** `density?: 'base' | 'compact'`

Changes the underlying values of size.

This allows us to use the same components in different contexts, like in cards.

```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon density="compact" size="sm" name="circle" />
```
