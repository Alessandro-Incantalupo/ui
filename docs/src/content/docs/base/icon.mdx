### Props
Extends [Tag](/utility/tag).

```ts
name?: tabler-icon-name
```
For all available icons see https://tablericons.com/.

### Defaults
```ts
as = 'i'
```

## Examples

### Color
```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon name="friends" />
<Icon name="friends" color="accent" />
```

### Size
```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon name="friends" size="lg" />
<Icon name="friends" size="md" />
<Icon name="friends" size="sm" />
```

### Compact
Why size and compact? Read [the page about sizes](/theme/size).
```astro live
---
import { Icon } from '@fullui/ui'
---
<Icon name="friends" size="lg" compact />
<Icon name="friends" size="md" compact />
<Icon name="friends" size="sm" compact />
```
