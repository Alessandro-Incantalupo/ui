An icon or a component that looks like a icon.

Currently only [Tabler Icons](https://tablericons.com/) are supported. The component does however work without any aditional setup, this component simply inlines a fully customizable SVG.

If you want to use different icons, we advise you to use [astro-icon](https://github.com/natemoo-re/astro-icon#readme).

<br />

```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon name="circle" />
```

## Props

### `name`

**Type:** `name?: [TABLER ICON NAME]`

For all available icons see https://tablericons.com/.

```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon name="circle" />
<Icon name="arrow-right" />
<Icon name="brand-x" />
<Icon name="brand-github" />
```

### `size`

<SizeProp />

```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon size="sm" name="circle" />
<Icon size="md" name="circle" />
<Icon size="lg" name="circle" />
```

### `density`

<DensityProp />

```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon density="compact" size="sm" name="circle" />
<Icon density="compact" size="md" name="circle" />
<Icon density="compact" size="lg" name="circle" />
```

### `variant`

**Type:** `color?: 'base' | 'muted'`

<VariantProp />

```astro live

<Icon variant="muted" name="circle" />
```
