---
title: Icon
description: Displays a button or a component that looks like a button.
sidebar:
  order: 4
---

## Props
Extends [Tag](/components/tag).
```ts
name: string | null | undefined
```

## Defaults
```ts
as = 'i'
```

## Examples
Below examples use regular syntax, but work with [all syntaxes](/guides/components#syntax).

### Color
```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon name="friends" />
<Icon name="friends" color="accent" />
```

### Size
```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon name="friends" size="3" />
<Icon name="friends" size="2" />
<Icon name="friends" size="1" />
```

### Compact
*It might seem weird having sizes and also compact, but it has a lot of benefits. More on this later*
```astro live
---
import { Icon } from '@fullui/ui'
---
<Icon name="friends" size="3" compact />
<Icon name="friends" size="2" compact />
<Icon name="friends" size="1" compact />
```
