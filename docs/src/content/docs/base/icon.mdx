Displays a icon or a component that looks like a icon.

:::note
This component is currenty in beta.
If you find any problems or bugs report them here: [GitHub Issues](https://github.com/fullui/ui/issues).
:::

## Props
Extends [Tag](/utility/tag).
```ts
name?: string
```

## Defaults
```ts
as = 'i'
```

## Examples
Below examples use regular syntax, but work with [all syntaxes](/guides/components#syntax).

### Color
```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon name="friends" />
<Icon name="friends" color="accent" />
```

### Size
```astro live
---
import { Icon } from '@fullui/ui'
---

<Icon name="friends" size="large" />
<Icon name="friends" size="medium" />
<Icon name="friends" size="small" />
```

### Compact
*It might seem weird having sizes and also compact, but it has a lot of benefits. More on this later*
```astro live
---
import { Icon } from '@fullui/ui'
---
<Icon name="friends" size="large" compact />
<Icon name="friends" size="medium" compact />
<Icon name="friends" size="small" compact />
```
