Displays a logo image.

## Props
extends [Tag](/utility/tag) and [Image](/components/image).
```ts
src?: string
href?: string
```

## Defaults
```ts
as = 'a'
```

## Examples
The examples below could also be written with [different syntax](/overview/syntax).

```astro live
---
import { Logo, Image } from '@fullui/ui'
---

<Logo>
  <Image src="/./favicon/android-chrome-192x192.png" />
</Logo>
```
