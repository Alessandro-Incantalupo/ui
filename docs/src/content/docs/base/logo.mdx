Displays a logo image.

:::note
This component is currenty in beta.
If you find any problems or bugs report them here: [GitHub Issues](https://github.com/fulldevlabs/fullui/issues).
:::

## Props
extends [Tag](/components/tag) and [Image](/components/image).
```ts
src: string | null | undefined
href: string | null | undefined
```

## Defaults
```ts
as = 'a'
```

## Example
The example down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).

```astro live
---
import { Logo } from '@fullui/ui'
---

<Logo src="/./favicon.png" />
```
