Displays a logo image.

## Props
extends [Tag](/utility/tag) and [Image](/components/image).
```ts
src?: string
href?: string
```

## Defaults
```ts
as = 'a'
```

## Example
The example below uses regular syntax, check out [Syntax](/overview/syntax) for more options.

```astro live
---
import { Logo, Image } from '@fullui/ui'
---

<Logo>
  <Image src="/./favicon.png" />
</Logo>
```
