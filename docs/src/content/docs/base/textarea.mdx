## Props
Below props are available in addition to all [Tag](/foundation/tag) props.

### `placeholder`
```astro live
---
import { Textarea } from '@fullui/ui'
---

<Textarea name="message" placeholder="Name" />
```

### `size`
**Type:** `variant?: 'sm' | 'md' | 'lg'`
```astro live
---
import { Textarea } from '@fullui/ui'
---

<Textarea size="sm" name="message" placeholder="name" />
<Textarea size="md" name="message" placeholder="name" />
<Textarea size="lg" name="message" placeholder="name" />
```

### `compact`
**Type:** `compact?: boolean`
```astro live
---
import { Textarea } from '@fullui/ui'
---

<Textarea compact size="sm" name="message" placeholder="name" />
<Textarea compact size="md" name="message" placeholder="name" />
<Textarea compact size="lg" name="message" placeholder="name" />
```
