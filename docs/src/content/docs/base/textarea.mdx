A textarea element.

<br/>

```astro live
---
import { Textarea } from '@fullui/ui'
---

<Textarea name="message" placeholder="Message" />
```

## Props

### `size`
**Type:** `variant?: 'sm' | 'md' | 'lg'`
```astro live
---
import { Textarea } from '@fullui/ui'
---

<Textarea size="sm" name="message" placeholder="name" />
<Textarea size="md" name="message" placeholder="name" />
<Textarea size="lg" name="message" placeholder="name" />
```

### `density`
**Type:** `density?: 'base' | 'compact'`

Changes the underlying values of size.

This allows us to use the same components in different contexts, like in cards.

```astro live
---
import { Textarea } from '@fullui/ui'
---

<Textarea density="compact" size="sm" name="message" placeholder="name" />
```
