Renders a list with items

## Props
extends [Tag](/utility/tag) and [Children](/utility/children).

```ts
direction?: 'row' | 'column'
align?: 'stretch' | 'start' | 'center' | 'end'
```

## Defaults
```ts
as = 'ul'
```

## Example
The example below uses regular syntax, check out [Syntax](/overview/syntax) for more options.

```astro live
---
import { List, Item } from '@fullui/ui'
---

<List direction="column">
  <Item>Item</Item>
  <Item>Item</Item>
  <Item>Item</Item>
</List>
```
