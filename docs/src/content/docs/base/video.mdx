Displays a YouTube video.

## Props
extends [Tag](/utility/tag).
```ts
src?: string
thumbnail: any
```

## Example
The example below uses regular syntax, check out [Syntax](/overview/syntax) for more options.

:::note
This component only accepts `youtube.com` and `youtu.be` urls.
:::

```astro live
---
import { Video } from '@fullui/ui'
---

<Video src="https://youtu.be/dQw4w9WgXcQ?si=SSmA4Z0_wrwor6Xg" />
```
