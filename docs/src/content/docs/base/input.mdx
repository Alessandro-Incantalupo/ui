## Props
Below props are available in addition to all [Tag](/foundation/tag) props.

### `placeholder`
```astro live
---
import { Input } from '@fullui/ui'
---

<Input name="name" placeholder="Name" />
```

### `type`
```astro live
---
import { Input } from '@fullui/ui'
---

<Input name="email" placeholder="email" type="email" />
<Input name="password" placeholder="password" type="password" />
<Input name="number" placeholder="number" type="number" />
```

### `size`
**Type:** `variant?: 'sm' | 'md' | 'lg'`
```astro live
---
import { Input } from '@fullui/ui'
---

<Input size="sm" name="name" placeholder="name" />
<Input size="md" name="name" placeholder="name" />
<Input size="lg" name="name" placeholder="name" />
```

### `compact`
**Type:** `compact?: boolean`
```astro live
---
import { Input } from '@fullui/ui'
---

<Input compact size="sm" name="name" placeholder="name" />
<Input compact size="md" name="name" placeholder="name" />
<Input compact size="lg" name="name" placeholder="name" />
```
