Displays an input field.

## Props
extends [Tag](/utility/tag).
```ts
name?: string
```

## Defaults
```ts
as = 'input'
```

## Example
The example below uses regular syntax, check out [Syntax](/overview/syntax) for more options.

```astro live
---
import { Input } from '@fullui/ui'
---

<Input name="password" size="large" placeholder="Password" type="password" />
```
