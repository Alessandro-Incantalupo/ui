A native, styled input element.

<br/>

```astro live
---
import { Input } from '@fullui/ui'
---

<Input name="name" placeholder="Name" />
```

## Props

### `size`
**Type:** `variant?: 'sm' | 'md' | 'lg'`
```astro live
---
import { Input } from '@fullui/ui'
---

<Input size="sm" name="name" placeholder="name" />
<Input size="md" name="name" placeholder="name" />
<Input size="lg" name="name" placeholder="name" />
```

### `density`
**Type:** `density?: 'base' | 'compact'`

Changes the underlying values of size.

This allows us to use the same components in different contexts, like in cards.

```astro live
---
import { Input } from '@fullui/ui'
---

<Input density="compact" size="sm" name="name" placeholder="name" />
```
