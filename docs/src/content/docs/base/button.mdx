---
title: Button
description: Displays a button or a component that looks like a button.
---

### Props
Extends [Tag](/utility/tag), [Children](/utility/children)
```ts
variant?: 'solid' | 'soft' | 'surface' | 'outline' | 'ghost'
```

### Defaults
```ts
as = 'a'
variant = 'solid'
```

## Examples
The examples below could also be written with [different syntax](/overview/components).

### Variants
```astro live
---
import { Button } from '@fullui/ui'
---

<Button variant="primary">Button</Button>
<Button variant="secondary">Button</Button>
<Button variant="bordered">Button</Button>
<Button variant="ghost">Button</Button>
```

### Icon
```astro live
---
import { Button } from '@fullui/ui'
import { Icon } from '@fullui/ui'
---

<Button variant="secondary" icon.name="arrow-right" />
<Button variant="secondary" html="Button" icon.name="arrow-right" />
<Button variant="secondary" icon={{name: "arrow-left"}} html="Button"/>
```

### Size
```astro live
---
import { Button } from '@fullui/ui'
---

<Button variant="secondary" size="lg">Button</Button>
<Button variant="secondary" size="md">Button</Button>
<Button variant="secondary" size="sm">Button</Button>
```

### Compact
Why size and compact? Read [the page about sizes](/theme/size).
```astro live
---
import { Button } from '@fullui/ui'
---

<Button variant="secondary" compact size="lg">Button</Button>
<Button variant="secondary" compact size="md">Button</Button>
<Button variant="secondary" compact size="sm">Button</Button>
```
