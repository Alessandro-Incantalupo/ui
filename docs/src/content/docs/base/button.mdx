A button or a component that looks like a button.

<br/>

```astro live
---
import { Button } from '@fullui/ui'
---

<Button color="primary">Button</Button>
```

## Props
### `color`
**Type:** `color?: 'primary' | 'secondary' | 'outline' | 'ghost' = 'primary'`
```astro live
---
import { Button } from '@fullui/ui'
---

<Button color="primary">Button</Button>
<Button color="secondary">Button</Button>
<Button color="raised">Button</Button>
<Button color="base">Button</Button>
<Button color="lowered">Button</Button>
```

### `size`
**Type:** `variant?: 'sm' | 'md' | 'lg'`
```astro live
---
import { Button } from '@fullui/ui'
---

<Button size="sm">Button</Button>
<Button size="md">Button</Button>
<Button size="lg">Button</Button>
```

### `icon`
**Type:** `icon?: Icon | Icon['name']`

Either an icon object or a string representing the icon name.

```astro live
---
import { Button } from '@fullui/ui'
---

<Button icon="circle" text="Button"/>
<Button text="Button" icon="circle" />
<Button icon={{name: 'circle', class: 'my-icon'}} />
```

All of the [`icon`](/base/icon) component props are accepted here as well.

### `density`
**Type:** `density?: 'base' | 'compact'`

Changes the underlying values of size.

This allows us to use the same components in different contexts, like in cards.

```astro live
---
import { Button } from '@fullui/ui'
---

<Button density="compact" size="sm">Button</Button>
```
