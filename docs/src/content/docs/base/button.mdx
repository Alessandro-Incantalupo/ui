## Props
Below props are available in addition to all [Tag](/foundation/tag) and [Children](/foundation/children) props.

### `color`
**Type:** `color?: 'primary' | 'secondary' | 'outline' | 'ghost' = 'primary'`
```astro live
---
import { Button } from '@fullui/ui'
---

<Button color="primary">Button</Button>
<Button color="secondary">Button</Button>
<Button color="raised">Button</Button>
<Button color="base">Button</Button>
<Button color="lowered">Button</Button>
```

### `size`
**Type:** `variant?: 'sm' | 'md' | 'lg'`
```astro live
---
import { Button } from '@fullui/ui'
---

<Button size="sm">Button</Button>
<Button size="md">Button</Button>
<Button size="lg">Button</Button>
```

### `compact`
**Type:** `compact?: boolean`
```astro live
---
import { Button } from '@fullui/ui'
---

<Button compact size="sm">Button</Button>
<Button compact size="md">Button</Button>
<Button compact size="lg">Button</Button>
```

### `icon`
**Type:** `icon?: boolean`
```astro live
---
import { Button } from '@fullui/ui'
---

<Button icon.name="circle" />
<Button icon.name="circle" text="Button"/>
<Button text="Button" icon.name="circle" />
```
