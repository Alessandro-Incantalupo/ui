Displays a button or a component that looks like a button.

### Props
Extends [Tag](/utility/tag), [Children](/utility/children)
```ts
variant?: 'solid' | 'soft' | 'surface' | 'outline' | 'ghost'
```

### Defaults
```ts
as = 'a'
variant = 'solid'
color = 'accent'
```

## Examples

### Variants
```astro live
---
import { Button } from '@fullui/ui'
---

<Button variant="solid">Button</Button>
<Button variant="soft">Button</Button>
<Button variant="surface">Button</Button>
<Button variant="outline">Button</Button>
<Button variant="ghost">Button</Button>
```

### Color
```astro live
---
import { Button } from '@fullui/ui'

<Button color="base" variant="soft">Button</Button>

<Button color="base" variant="outline">Button</Button>
```
