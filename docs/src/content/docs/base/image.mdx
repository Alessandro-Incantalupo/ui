Displays an image.

## Props
extends [Tag](/utility/tag).
```ts
src?: string
alt?: string
loading?: string
mask:? boolean
sizes?: string
position?: string
frame?: string
```

## Defaults
```ts
as = 'img'
```

## Example

```astro live
---
import { Image } from '@fullui/ui'
---

<Image 
  src="https://picsum.photos/id/237/536/354" 
  alt="Example image" 
  loading="lazy" 
/>
```
