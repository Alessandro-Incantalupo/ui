Displays an image.

:::danger
This component is currenty a work in progress and not fully functional. 
If you find any problems or bugs report them here: [GitHub Issues](https://github.com/fullui/ui/issues).
:::

## Props
extends [Tag](/utility/tag).
```ts
src?: string
alt?: string
loading?: string
mask:? boolean
sizes?: string
position?: string
frame?: string
```

## Defaults
```ts
as = 'img'
```

## Example
The example down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).

```astro live
---
import { Image } from '@fullui/ui'
---

<Image 
  src="https://picsum.photos/id/237/536/354" 
  alt="Example image" 
  loading="lazy" 
/>
```
