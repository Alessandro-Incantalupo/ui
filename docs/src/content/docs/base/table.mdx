### Formats

#### `minimal`

Build a table with the minimum amount of code and components.

```astro live
---
import { Table } from 'fulldev-ui'
---

<Table
  header={['First Name', 'Last Name', 'Age']}
  body={[
    ['John', 'Doe', '31'],
    ['Jane', 'Doe', '29'],
  ]}
  footer={[{ text: 'Total', colspan: '2' }, '2']}
/>
```

#### `medium`

Build a table with a little more code and components.

```astro live
---
import {
  Table,
  TableBody,
  TableFooter,
  TableHeader,
  TableRow,
} from 'fulldev-ui'
---

<Table>
  <TableHeader cells={['First Name', 'Last Name', 'Age']} />
  <TableBody>
    <TableRow cells={['John', 'Doe', '31']} />
    <TableRow cells={['Jane', 'Doe', '29']} />
  </TableBody>
  <TableFooter cells={[{ text: 'Total', colspan: '2' }, '2']} />
</Table>
```

:::note

instead of creating each row with a `TableRow` component, you can use the `rows` prop in the `TableBody` component.

```astro
<TableBody
  rows={[
    ['John', 'Doe', '31'],
    ['Jane', 'Doe', '29'],
  ]}
/>
```

:::

#### `advanced`

Build a table completely with each child component.

```astro live
---
import {
  Table,
  TableBody,
  TableCell,
  TableFooter,
  TableHead,
  TableHeader,
  TableRow,
} from 'fulldev-ui'
---

<Table>
  <TableHeader>
    <TableRow>
      <TableHead>First Name</TableHead>
      <TableHead>Last Name</TableHead>
      <TableHead>Age</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell>John</TableCell>
      <TableCell>Doe</TableCell>
      <TableCell>31</TableCell>
    </TableRow>
    <TableRow>
      <TableCell>Jane</TableCell>
      <TableCell>Doe</TableCell>
      <TableCell>29</TableCell>
    </TableRow>
  </TableBody>
  <TableFooter>
    <TableRow>
      <TableCell colspan="2">Total</TableCell>
      <TableCell>2</TableCell>
    </TableRow>
  </TableFooter>
</Table>
```
