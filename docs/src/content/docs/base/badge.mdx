Displays a Badge or a component that looks like a Badge.

## Props
Extends [Tag](/utility/tag), [Children](/utility/children) and [Icon](/components/icon).
```ts
variant: 'solid' | 'soft' | 'surface' | 'outline' | null | undefined
```

## Defaults
```ts
tag = 'span'
variant = 'solid'
color = 'accent'
```

## Examples
Below examples use regular props, but also accept [Endomorphic props](/guides/macromorphic).

### Variants
```astro live
---
import { Badge } from '@fullui/ui'
---

<Badge variant="solid">Badge</Badge>
<Badge variant="soft">Badge</Badge>
<Badge variant="surface">Badge</Badge>
<Badge variant="outline">Badge</Badge>
```

### Color
```astro live
---
import { Badge } from '@fullui/ui'
---

<Badge color="accent" variant="solid">Badge</Badge>
<Badge color="accent" variant="soft">Badge</Badge>
<Badge color="accent" variant="surface">Badge</Badge>
<Badge color="accent" variant="outline">Badge</Badge>
```

### Contrast
```astro live
---
import { Badge } from '@fullui/ui'
---

<Badge contrast variant="solid">Badge</Badge>
<Badge contrast variant="soft">Badge</Badge>
<Badge contrast variant="surface">Badge</Badge>
<Badge contrast variant="outline">Badge</Badge>
```

### Icon
```astro live
---

import { Icon } from '@fullui/ui'
---

<Badge icon={{name: "arrow-right"}}></Badge>

<Badge icon={{name: "arrow-right"}}>Badge</Badge>
```

### Size

