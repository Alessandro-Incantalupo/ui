### Props
Extends [Tag](/utility/tag), [Children](/utility/children)
```ts
variant?: 'solid' | 'soft' | 'surface' | 'outline'
```

### Defaults
```ts
as = 'span'
variant = 'solid'
```

## Examples
The examples below could also be written with [different syntax](/overview/components).

### Variants
```astro live
---
import { Badge } from '@fullui/ui'
---

<Badge variant="solid">Badge</Badge>
<Badge variant="soft">Badge</Badge>
<Badge variant="surface">Badge</Badge>
<Badge variant="outline">Badge</Badge>
```
