A chip useful for displaying a small amount of information.

<br/>

```astro live
---
import { Badge } from '@fullui/ui'
---

<Badge color="primary">Badge</Badge>
```

## Props

### `html`
**Type:** `html?: string`
```astro live
---
import { Badge } from '@fullui/ui'
---

<Badge html="<strong>Badge</strong>"/>
```

### `text`
**Type:** `text?: string`
```astro live
---
import { Badge } from '@fullui/ui'
---

<Badge text="Badge"/>
```

### `icon`
**Type:** `icon?: Icon | Icon['name']`

Either an icon object or a string representing the icon name.

```astro live
---
import { Badge } from '@fullui/ui'
---

<Badge icon="circle" text="Badge"/>
<Badge text="Badge" icon="circle" />
```

### `size`
**Type:** `variant?: 'sm' | 'md' | 'lg'`
```astro live
---
import { Badge } from '@fullui/ui'
---

<Badge size="sm">Badge</Badge>
<Badge size="md">Badge</Badge>
<Badge size="lg">Badge</Badge>
```

### `density`
**Type:** `density?: 'base' | 'compact'`

Changes the underlying values of size.

This allows us to use the same components in different contexts, like in cards.

```astro live
---
import { Badge } from '@fullui/ui'
---

<Badge density="compact" size="sm">Badge</Badge>
```

### `color`
**Type:** `color?: 'primary' | 'secondary' | 'outline' | 'ghost' = 'primary'`
```astro live
---
import { Badge } from '@fullui/ui'
---

<Badge color="primary">Badge</Badge>
<Badge color="secondary">Badge</Badge>
<Badge color="raised">Badge</Badge>
<Badge color="base">Badge</Badge>
<Badge color="lowered">Badge</Badge>
```
