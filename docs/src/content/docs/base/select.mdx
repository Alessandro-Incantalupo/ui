---
title: Select
description: Displays a selection with options.
---

## Props
Below props are available in addition to all [Tag](/foundation/tag) props.

### `placeholder`
```astro live
---
import { Input } from '@fullui/ui'
---

<Input name="name" placeholder="Name" />
```

### `type`
```astro live
---
import { Input } from '@fullui/ui'
---

<Input name="email" placeholder="email" type="email" />
<Input name="password" placeholder="password" type="password" />
<Input name="number" placeholder="number" type="number" />
```

### `size`
**Type:** `variant?: 'sm' | 'md' | 'lg'`
```astro live
---
import { Select } from '@fullui/ui'

const options = [
  {
    component: 'Option',
    value: 'option 1',
    text: 'Option 1'
  },
  {
    component: 'Option',
    value: 'option 2',
    text: 'Option 2'
  },
  {
    component: 'Option',
    value: 'option 3',
    text: 'Option 3'
  }
]
---

<Select size="sm" {options}/>
<Select size="md" {options}/>
<Select size="lg" {options}/>
```

### `compact`
**Type:** `compact?: boolean`
```astro live
---
import { Select } from '@fullui/ui'

const options = [
  {
    component: 'Option',
    value: 'option 1',
    text: 'Option 1'
  },
  {
    component: 'Option',
    value: 'option 2',
    text: 'Option 2'
  },
  {
    component: 'Option',
    value: 'option 3',
    text: 'Option 3'
  }
]
---

<Select compact size="sm" {options}/>
<Select compact size="md" {options}/>
<Select compact size="lg" {options}/>
```
