---
title: Select
description: Displays a selection with options.
---

## Props
Below props are available in addition to all [Tag](/foundation/tag) props.

#### `placeholder`
```astro live
---
import { Select } from '@fullui/ui'

const options = [
  {
    value: 'option 1',
    text: 'Option 1'
  },
  {
    value: 'option 2',
    text: 'Option 2'
  }
]
---

<Select name="name" placeholder="Name" {options} />
```

#### `size`
**Type:** `variant?: 'sm' | 'md' | 'lg'`
```astro live
---
import { Select } from '@fullui/ui'

const options = [
  {
    value: 'option 1',
    text: 'Option 1'
  },
  {
    value: 'option 2',
    text: 'Option 2'
  }
]
---

<Select size="sm" {options}/>
<Select size="md" {options}/>
<Select size="lg" {options}/>
```

#### `compact`
**Type:** `compact?: boolean`
```astro live
---
import { Select } from '@fullui/ui'

const options = [
  {
    value: 'option 1',
    text: 'Option 1'
  },
  {
    value: 'option 2',
    text: 'Option 2'
  },
]
---

<Select density="compact" size="sm" {options} />
<Select density="compact" size="md" {options} />
<Select density="compact" size="lg" {options} />
```


## Props
extends [Tag](/foundation/tag).
