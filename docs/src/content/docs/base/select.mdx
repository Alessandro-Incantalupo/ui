A native but styled select element with options to pick from. 

<br/>

```astro live
---
import { Select, SelectOption } from '@fullui/ui'
---

<Select>
  <SelectOption>Option 1</SelectOption>
  <SelectOption>Option 2</SelectOption>
</Select>
```

## Props
### `options`
**Type:** `([SelectOption] | string)[]`

```astro live
---
import { Select } from '@fullui/ui'
---

<Select
  options={[
    {
      text: 'Option 1',
      value: 'value 1',
    },
    'Option 2'
  ]}
/>
```

#### option > `text`
Text to render inside the component.

#### option > HTML Attributes
All HTML `<option/>` attributes are accepted as props.

### `size`
**Type:** `variant?: 'sm' | 'md' | 'lg'`
```astro live
---
import { Select } from '@fullui/ui'
---

<Select size="sm" options={['Option 1', 'Option 2']} />
<Select size="md" options={['Option 1', 'Option 2']} />
<Select size="lg" options={['Option 1', 'Option 2']} />
```

### `compact`
**Type:** `compact?: boolean`
```astro live
---
import { Select } from '@fullui/ui'
---

<Select size="sm" options={['Option 1', 'Option 2']} />
<Select size="md" options={['Option 1', 'Option 2']} />
<Select size="lg" options={['Option 1', 'Option 2']} />
```

### HTML Attributes
All HTML `<select/>` attributes are accepted as props.
