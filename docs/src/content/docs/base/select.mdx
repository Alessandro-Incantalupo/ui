A native but styled select element with options to pick from. 

<br/>

```astro live
---
import { Select, SelectOption } from '@fullui/ui'
---

<Select>
  <SelectOption>Option 1</SelectOption>
  <SelectOption>Option 2</SelectOption>
</Select>
```

## Props
Below are the props available to this component.

### `options`
**Type:** `Array<{ text: string, value: string }>`
```astro live
---
import { Select } from '@fullui/ui'
---

<Select
  options={[
    {
      text: 'Option 1',
      value: 'value 1',
    },
    {
      text: 'Option 2',
      value: 'value 2',
    },
  ]}
/>
```

#### `size`
**Type:** `variant?: 'sm' | 'md' | 'lg'`
```astro live
---
import { Select } from '@fullui/ui'

const options = [
  {
    value: 'option 1',
    text: 'Option 1'
  },
  {
    value: 'option 2',
    text: 'Option 2'
  }
]
---

<Select size="sm" {options}/>
<Select size="md" {options}/>
<Select size="lg" {options}/>
```

#### `compact`
**Type:** `compact?: boolean`
```astro live
---
import { Select } from '@fullui/ui'

const options = [
  {
    value: 'option 1',
    text: 'Option 1'
  },
  {
    value: 'option 2',
    text: 'Option 2'
  },
]
---

<Select density="compact" size="sm" {options} />
<Select density="compact" size="md" {options} />
<Select density="compact" size="lg" {options} />
```
