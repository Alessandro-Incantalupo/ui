title: Variant

Variants are used to change the appearance of a component. They are often used to change the color, size, or shape of a component. Below are the possible values for this prop.

<br />

import { Card, Button } from 'fulldev-ui'

## Prop values

Prop values differ per component. Below are the possible values for this prop.

### `base`

<Card variant="base">Base Card</Card>

### `raised`

<Card variant="raised">Raised Card</Card>

### `primary`

<Card variant="primary">Primary Card</Card>

### `solid`

<Card>
  <Button variant="solid">Solid Button</Button>
</Card>

### `soft`

<Card>
  <Button variant="soft">Soft Button</Button>
</Card>

### `surface`

<Card>
  <Button variant="surface">Surface Button</Button>
</Card>

### `outline`

<Card>
  <Button variant="outline">Outline Button</Button>
</Card>

### `ghost`

<Card>
  <Button variant="ghost">Ghost Button</Button>
</Card>

### `link`

<Card>
  <Button variant="link">Link Button</Button>
</Card>
