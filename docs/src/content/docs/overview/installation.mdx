Lets not make it more steps than neccesary.

Just install the package and import the components, no integration needed!

<Tabs>
  <TabItem label="pnpm">
    ```bash
    pnpm add @fullui/ui
    ```

  </TabItem>

    <TabItem label="yarn">
    ```bash
    yarn add @fullui/ui
    ```

  </TabItem>

    <TabItem label="npm">
    ```bash
    npm i @fullui/ui
    ```

  </TabItem>

    <TabItem label="bun">
    ```bash
    bun i @fullui/ui
    ```

  </TabItem>
</Tabs>
