Run one of the following commands to install Fullui into your project:

<Tabs>
  <TabItem label="pnpm">
    ```bash
    pnpm add @fullui/ui
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add @fullui/ui
    ```
  </TabItem>
  <TabItem label="npm">
    ```bash
    npm install @fullui/ui
    ```
  </TabItem>
</Tabs>

#### Integration
The design system is enabled via a simple Astro integration:

```ts
// astro.config.ts
import { defineConfig } from 'astro/config'
import fullui from '@fullui/ui/integration'

export default defineConfig({
  integrations: [
    fullui(),
  ],
})
```
