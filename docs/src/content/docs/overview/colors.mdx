Setting colors is done using CSS variables.

This way we can leverage the power of cascading, which means that a size applies to all children as well.

For example; we can set a size on a bocks and have everything inside it adapt to this theme; text colors, buttons, backgrounds, etc.

## Themes

We use the variable names `background`, `hover`, `border`, `focus` and `foreground`.

```css
:root {
  --base-background: hsl(0, 0%, 100%);
  --base-hover: hsl(0, 0%, 97%);
  --base-border: hsl(0, 0%, 83%);
  --base-focus: hsl(0, 0%, 10%);
  --base-foreground: hsl(0, 0%, 4%);
}
```

#### Dark theme

Dark mode is applied by simply swapping the colors in a `.full-theme-dark` class.

```css
.full-theme-dark {
  --base-background: #020202;
  --base-hover: #101010;
  --base-border: #333333;
  --base-focus: #fff;
  --base-foreground: #fff;
}
```

#### Custom themes

You can create as many themes as you want, using classes with prefix `full-theme-`.

```css
.full-theme-dark {
  --base-background: lightblue;
  --base-hover: blue;
  --base-border: blue;
  --base-focus: blue;
  --base-foreground: darkblue;
}
```

## Variants

Using variants, we can swap those 5 colors for different ones. The following variants are available.

- **raised:** used for cards and other panels
- **accent:** used for accent colored text, like a tagline
- **muted:** used for lower contrast text elements
- **ghost:** used for interactive components without a background and border
- **outline:** used for interactive components with a border but no background
- **secondary:** used for secondary interactive components
- **primary:** used for primary interactive components

While you **can** set each 5 colors for each variant, you **dont have to**.

We advise you to only set the following, and the rest will be determined automatically.

```
--base-background: hsl(0, 0%, 100%);
--base-hover: hsl(0, 0%, 97%);
--base-border: hsl(0, 0%, 83%);
--base-focus: hsl(0, 0%, 10%);
--base-foreground: hsl(0, 0%, 4%);

--raised-background: hsl(0, 0%, 97%);

--muted-foreground: hsl(0, 0%, 25%);

--outline-border: hsl(0, 0%, 83%);

--secondary-background: hsl(0, 0%, 94%);
--secondary-hover: hsl(0, 0%, 92%);
--secondary-foreground: var(--base-foreground);

--primary-background: hsl(0, 0%, 0%);
--primary-hover: hsl(0, 0%, 16%);
--primary-foreground: hsl(0, 0%, 100%);
```

For all default and avaiable variables, see the `colors.css` file in the repository.
