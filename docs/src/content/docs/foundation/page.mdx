### Props
Extends [Tag](/foundation/tag). Uses [Head](/utility/head), [Body](/utility/body)
```ts
head: ComponentProps<typeof Head>
body: ComponentProps<typeof Body>
```

### Defaults
```ts
as = 'html' 
```

## Example
_This example renders differently in this example, because this website already has a head._
```astro live
---
import { Page } from '@fullui/ui'
---

<Page 
  head={{
    title: 'Title',
    description: 'Description',
  }}
  body={{
    heading: {
      html: 'I am in a body',
    }
  }}
/>
```
