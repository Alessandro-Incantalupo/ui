All wrapper components are subject to breaking changes. Thet are not yet ready for production use.
:::

A general wrapper component which sets:

- Outer margins and paddings
- Layout inside the wrapper
- Spacing between child elements

<br />

```astro live
---
import { Button, Group } from 'fulldev-ui'
---

<Group>
  <Button> Button </Button>
  <Button> Button </Button>
</Group>
```

## Props

### `direction`

**Type:** `direction?: 'vertical' | 'horizontal' = 'vertical'`

```astro live
---
import { Button, Group } from 'fulldev-ui'
---

<Group direction="horizontal">
  <Button> Button </Button>
  <Button> Button </Button>
</Group>
```

### `x`

**Type:** `x?: 'left' | 'center' | 'right' | 'spread' | 'fill' = 'left'`
{/* TODO kan ook voor deck! spreak is grid met ruimtes, fill is cover, top is masonry rechte line top, bottom bottom etc */}

```astro live
---
import { Button, Group, Heading } from 'fulldev-ui'
---

<Group x="left">
  <Heading>Lorem ipsum dolor </Heading>
  <Button>Button</Button>
</Group>
<Group direction="horizontal" x="spread">
  <Heading>Lorem ipsum dolor </Heading>
  <Button>Button</Button>
</Group>
<Group x="fill">
  <Heading>Lorem ipsum dolor </Heading>
  <Button>Button</Button>
</Group>
```

### `y`

**Type:** `y?: 'top' | 'center' | 'bottom' | 'spread' | 'fill' = 'top'`

Use this setting in combination with direction `horizontal`.
