Astro Subtext component for a variety of subtext tags, like 'p' and more

### Props
extends [Tag](/components/tag) and [Children](/components/children).

### Defaults
```ts
as = 'p' 
```

## Examples
### Color
```astro live
---
import { Subtext } from '@fullui/ui'
---

<Subtext>The quick brown fox jumps over the lazy dog</Subtext=>   
<Subtext color="accent">The quick brown fox jumps over the lazy dog</Subtext=>  
```

### Contrast
```astro live
---
import { Subtext } from '@fullui/ui'
---

<Subtext contrast={true}>The quick brown fox jumps over the lazy dog</Subtext=>   
<Subtext contrast={true} color="accent">The quick brown fox jumps over the lazy dog</Subtext=>  
```

### Size
```astro live
---
import { Subtext } from '@fullui/ui'
--- 
<Subtext size="3">The quick brown fox jumps over the lazy dog</Subtext>   
<Subtext size="2">The quick brown fox jumps over the lazy dog</Subtext>  
<Subtext size="1">The quick brown fox jumps over the lazy dog</Subtext>   
```

### Compact
*It might seem weird having sizes and also compact, but it has a lot of benefits. More on this later*
```astro live
---
import { Subtext } from '@fullui/ui'
--- 
<Subtext size="3" compact>The quick brown fox jumps over the lazy dog</Subtext>   
<Subtext size="2" compact>The quick brown fox jumps over the lazy dog</Subtext>  
<Subtext size="1" compact>The quick brown fox jumps over the lazy dog</Subtext>   
```
