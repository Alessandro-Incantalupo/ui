Astro Tagline component for a variety of tagline tags, like 'p' and more

### Props
Extends [Tag](/utility/tag), [Children](/utility/children).

### Defaults
```ts
as = 'p' 
```

## Examples
The examples below could also be written with [different syntax](/overview/components).

### Color
```astro live
---
import { Tagline } from '@fullui/ui'
---

<Tagline>The quick brown fox jumps over the lazy dog</Tagline=>   
<Tagline color="accent">The quick brown fox jumps over the lazy dog</Tagline=>  
```

### Contrast
```astro live
---
import { Tagline } from '@fullui/ui'
---

<Tagline contrast={true}>The quick brown fox jumps over the lazy dog</Tagline=>   
<Tagline contrast={true} color="accent">The quick brown fox jumps over the lazy dog</Tagline=>  
```

### Size
```astro live
---
import { Tagline } from '@fullui/ui'
--- 
<Tagline size="lg">The quick brown fox jumps over the lazy dog</Tagline>   
<Tagline size="md">The quick brown fox jumps over the lazy dog</Tagline>  
<Tagline size="sm">The quick brown fox jumps over the lazy dog</Tagline>   
```

### Compact
Why size and compact? Read [the page about sizes](/theme/size).
```astro live
---
import { Tagline } from '@fullui/ui'
--- 
<Tagline size="lg" compact>The quick brown fox jumps over the lazy dog</Tagline>   
<Tagline size="md" compact>The quick brown fox jumps over the lazy dog</Tagline>  
<Tagline size="sm" compact>The quick brown fox jumps over the lazy dog</Tagline>   
```
