Astro Subtext component for a variety of subtext tags, like 'p' and more

### Props
Extends [Tag](/utility/tag), [Children](/utility/children).

### Defaults
```ts
as = 'p' 
```

## Examples
The examples below could also be written with [different syntax](/overview/components).

### Color
```astro live
---
import { Subtext } from '@fullui/ui'
---

<Subtext>The quick brown fox jumps over the lazy dog</Subtext=>   
<Subtext color="accent">The quick brown fox jumps over the lazy dog</Subtext=>  
```

### Contrast
```astro live
---
import { Subtext } from '@fullui/ui'
---

<Subtext contrast={true}>The quick brown fox jumps over the lazy dog</Subtext=>   
<Subtext contrast={true} color="accent">The quick brown fox jumps over the lazy dog</Subtext=>  
```

### Size
```astro live
---
import { Subtext } from '@fullui/ui'
--- 
<Subtext size="lg">The quick brown fox jumps over the lazy dog</Subtext>   
<Subtext size="md">The quick brown fox jumps over the lazy dog</Subtext>  
<Subtext size="sm">The quick brown fox jumps over the lazy dog</Subtext>   
```

### Compact
Why size and compact? Read [the page about sizes](/theme/size).
```astro live
---
import { Subtext } from '@fullui/ui'
--- 
<Subtext size="lg" compact>The quick brown fox jumps over the lazy dog</Subtext>   
<Subtext size="md" compact>The quick brown fox jumps over the lazy dog</Subtext>  
<Subtext size="sm" compact>The quick brown fox jumps over the lazy dog</Subtext>   
```
