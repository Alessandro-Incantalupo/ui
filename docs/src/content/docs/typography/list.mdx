A list element.

<br/>
  
```astro live
---
import { List, ListItem } from '@fullui/ui'
---

<List>
  <ListItem>Item 1</ListItem>
  <ListItem>Item 2</ListItem>
  <ListItem>Item 3</ListItem>
</List>
```

## Props
### `items`
#### `items > html`

#### `item > HTML Attributes`
All HTML `<li/>` attributes are accepted as props too.

### `size`
**Type:** `size?: 'sm' | 'md' | 'lg'`
```astro live
---
import { List, ListItem } from '@fullui/ui'
---

<List size="sm">
  <ListItem>Item 1</ListItem>
  <ListItem>Item 2</ListItem>
  <ListItem>Item 3</ListItem>
</List>
<List size="md">
  <ListItem>Item 1</ListItem>
  <ListItem>Item 2</ListItem>
  <ListItem>Item 3</ListItem>
</List>
<List size="lg">
  <ListItem>Item 1</ListItem>
  <ListItem>Item 2</ListItem>
  <ListItem>Item 3</ListItem>
</List>
```

### `density`
**Type:** `density?: 'base' | 'compact'`

Changes the underlying values of size.

This allows us to use the same components in different contexts, like in cards.

```astro live
---
import { List, ListItem } from '@fullui/ui'
---

<List density="compact">
  <ListItem>Item 1</ListItem>
  <ListItem>Item 2</ListItem>
  <ListItem>Item 3</ListItem>
</List>
```

### `color`
**Type:** `color?: 'primary' | 'secondary' | 'outline' | 'ghost' = 'primary'`
```astro live
---
import { List, ListItem } from '@fullui/ui'
---

<List color="primary">
  <ListItem>Item 1</ListItem>
  <ListItem>Item 2</ListItem>
  <ListItem>Item 3</ListItem>
</List>
```
