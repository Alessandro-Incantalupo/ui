## Props
Below props are available in addition to all [Tag](/foundation/tag) and [Children](/foundation/children) props.

### `color`
**Type:** `color?: 'underline'`
```astro live
---
import { List } from '@fullui/ui'
---

<List>List</List>
<List color="lowered">List</List>
```

### `size`
**Type:** `size?: 'sm' | 'md' | 'lg'`
```astro live
---
import { List } from '@fullui/ui'
---

<List size="sm">List</List>
<List size="md">List</List>
<List size="lg">List</List>
```

### `compact`
**Type:** `compact?: boolean`

In compact mode size `xs` is equal too `sm` and size `xl` / `2xl` are equal to `lg`
```astro live
---
import { List } from '@fullui/ui'
---

<List compact size="sm">List</List>
<List compact size="md">List</List>
<List compact size="lg">List</List>
```
