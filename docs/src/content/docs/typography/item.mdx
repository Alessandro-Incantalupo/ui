title: Item
---

## Props

#### `color`
**Type:** `color?: 'underline'`
```astro live
---
import { Item } from '@fullui/ui'
---

<Item>Item</Item>
<Item color="lowered">Item</Item>
```

#### `size`
**Type:** `size?: 'sm' | 'md' | 'lg'`
```astro live
---
import { Item } from '@fullui/ui'
---

<Item size="sm">Item</Item>
<Item size="md">Item</Item>
<Item size="lg">Item</Item>
```

#### `compact`
**Type:** `compact?: boolean`

In compact mode size `xs` is equal too `sm` and size `xl` / `2xl` are equal to `lg`
```astro live
---
import { Item } from '@fullui/ui'
---

<Item compact size="sm">Item</Item>
<Item compact size="md">Item</Item>
<Item compact size="lg">Item</Item>
```
