## Props
Below props are available in addition to all [Tag](/utility/tag) and [Children](/utility/children) props.

#### `color`
**Type:** `color?: 'underline'`
```astro live
---
import { Link } from '@fullui/ui'
---

<Link href="#">Link</Link>
<Link href="#" color="lowered">Link</Link>
```

#### `size`
**Type:** `size?: 'sm' | 'md' | 'lg'`
```astro live
---
import { Link } from '@fullui/ui'
---

<Link href="#" size="sm">Link</Link>
<Link href="#" size="md">Link</Link>
<Link href="#" size="lg">Link</Link>
```

#### `compact`
**Type:** `compact?: boolean`

In compact mode size `xs` is equal too `sm` and size `xl` / `2xl` are equal to `lg`
```astro live
---
import { Link } from '@fullui/ui'
---

<Link href="#" compact size="sm">Link</Link>
<Link href="#" compact size="md">Link</Link>
<Link href="#" compact size="lg">Link</Link>
```
