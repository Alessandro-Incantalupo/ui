---
title: Link
description: Astro link component for rendering anchor tags.
---

### Props
Extends [Tag](/utility/tag), [Children](/utility/children).

### Defaults
```ts
as = 'a'
```

## Examples
The examples below could also be written with [different syntax](/overview/syntax).

### Color
```astro live
---
import { Link } from '@fullui/ui'
---
<Link href="#">Link</Link>
<Link href="#" color="accent">Link</Link>
```

### Contrast
```astro live
---
import { Link } from '@fullui/ui'
---

<Link href="#" color="base" contrast>Link</Link>
<Link href="#" color="accent" contrast>Link</Link>
```

### Icon
```astro live
---
import { Link } from '@fullui/ui'
---

<Link href="#" icon={{name: "arrow-right"}}>Link</Link>
```

### Size
```astro live
---
import { Link } from '@fullui/ui'
---
<Link href="#" size="lg">Link</Link>
<Link href="#" size="md">Link</Link>
<Link href="#" size="sm">Link</Link>
```

### Compact
Why size and compact? Read [the page about sizes](/theme/size).
```astro live
---
import { Link } from '@fullui/ui'
---

<Link href="#" size="lg" compact>Link</Link>
<Link href="#" size="md" compact>Link</Link>
<Link href="#" size="sm" compact>Link</Link>
```
