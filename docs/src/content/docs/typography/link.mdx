A link element.

<br/>

```astro live
---
import { Link } from '@fullui/ui'
---

<Link href="#">Link</Link>
```

## Props
### `color`
**Type:** `color?: 'underline'`
```astro live
---
import { Link } from '@fullui/ui'
---

<Link href="#">Link</Link>
<Link href="#" color="lowered">Link</Link>
```

### `size`
**Type:** `size?: 'sm' | 'md' | 'lg'`
```astro live
---
import { Link } from '@fullui/ui'
---

<Link href="#" size="sm">Link</Link>
<Link href="#" size="md">Link</Link>
<Link href="#" size="lg">Link</Link>
```

<Theming/>

### HTML Attributes
All HTML `<a/>` attributes are accepted as props too.
