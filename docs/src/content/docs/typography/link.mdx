## Props
Below props are available in addition to all [Tag](/foundation/tag) and [Children](/foundation/children) props.

#### `color`
**Type:** `color?: 'underline'`
```astro live
---
import { Link } from '@fullui/ui'
---

<Link href="#">Link</Link>
<Link href="#" color="lowered">Link</Link>
```

#### `size`
**Type:** `size?: 'sm' | 'md' | 'lg'`
```astro live
---
import { Link } from '@fullui/ui'
---

<Link href="#" size="sm">Link</Link>
<Link href="#" size="md">Link</Link>
<Link href="#" size="lg">Link</Link>
```
