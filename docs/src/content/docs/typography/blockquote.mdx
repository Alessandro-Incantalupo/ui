Displays text in a blockqoute.

### Props
Extends [Tag](/utility/tag), [Children](/utility/children) and [Icon](/components/icon).
```ts
variant?: 'solid' | 'soft'
```

### Defaults
```ts
tag = 'span'
variant = 'solid'
color = 'accent'
```

## Examples
Below examples use regular props, but also accept [Endomorphic props](/guides/macromorphic).

### Variants
```astro live
---
import { Blockquote } from '@fullui/ui'
---

<Blockquote variant="solid">
  Lorem ipsum dolor sit amet, consectetur adipiscing elit, <br /> 
  sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
</Blockquote>

<Blockquote variant="soft">
  Lorem ipsum dolor sit amet, consectetur adipiscing elit, <br /> 
  sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
</Blockquote>
```
