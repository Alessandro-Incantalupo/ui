Displays text in a blockqoute.

### Props
Extends [Tag](/utility/tag), [Children](/utility/children)

### Defaults
```ts
tag = 'blockquote'
```

## Examples
The examples below use regular syntax, check out [Syntax](/overview/syntax) for more options.

### Variants
```astro live
---
import { Blockquote } from '@fullui/ui'
---

<Blockquote variant="solid">
  Lorem ipsum dolor sit amet, consectetur adipiscing elit, <br /> 
  sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
</Blockquote>

<Blockquote variant="soft">
  Lorem ipsum dolor sit amet, consectetur adipiscing elit, <br /> 
  sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
</Blockquote>
```
