A paragraph element.

<br/>

```astro live
---
import { Paragraph } from '@fullui/ui'
---

<Paragraph>The quick brown fox jumps over the lazy dog</Paragraph>
```

## Props
Below props are available in addition to all [Tag](/foundation/tag) and [Children](/foundation/children) props.

### `color`
**Type:** `color?: 'base' | 'lowered'`
```astro live
---
import { Paragraph } from '@fullui/ui'
---

<Paragraph color="base">The quick brown fox jumps over the lazy dog</Paragraph>
<Paragraph color="lowered">The quick brown fox jumps over the lazy dog</Paragraph>
```

### `size`
**Type:** `size?:  'sm' | 'md' | 'lg'`
```astro live
---
import { Paragraph } from '@fullui/ui'
---

<Paragraph size="sm">The quick brown fox jumps over the lazy dog</Paragraph>
<Paragraph size="md">The quick brown fox jumps over the lazy dog</Paragraph>
<Paragraph size="lg">The quick brown fox jumps over the lazy dog</Paragraph>
```

### `density`
**Type:** `density?: 'base' | 'compact'`

Changes the underlying values of size.

This allows us to use the same components in different contexts, like in cards.

```astro live
---
import { Paragraph } from '@fullui/ui'
---

<Paragraph density="compact" size="sm">The quick brown fox jumps over the lazy dog</Paragraph>
```
