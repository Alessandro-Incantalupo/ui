A label or something that looks like a label.

<br/>

```astro live
---
import { Label } from '@fullui/ui'
---

<Label>Label</Label>
```

## Props
### `html`
**Type:** `html?: string`
```astro live
---
import { Label } from '@fullui/ui'
---

<Label html="<i>Label</i>"/>
```

### `text`
**Type:** `text?: string`
```astro live
---
import { Label } from '@fullui/ui'
---

<Label text="Label"/>
```

### `size`
**Type:** `variant?: 'sm' | 'md' | 'lg'`
```astro live
---
import { Label } from '@fullui/ui'
---

<Label size="sm">Label</Label>
<Label size="md">Label</Label>
<Label size="lg">Label</Label>
```

### `density`
**Type:** `density?: 'base' | 'compact'`

Changes the underlying values of size.

This allows us to use the same components in different contexts, like in cards.

```astro live
---
import { Label } from '@fullui/ui'
---

<Label density="compact">Label</Label>
```

### `color`
**Type:** `color?: 'base' | 'lowered'`
```astro live
---
import { Label } from '@fullui/ui'
---

<Label color="base">Label</Label>
<Label color="lowered">Label</Label>
```
