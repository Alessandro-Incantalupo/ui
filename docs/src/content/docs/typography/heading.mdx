---
title: Heading
description: Astro heading component for rendering h1-h6 tags
---

## Props
extends [Tag](/components/tag) and [Children](/components/children).
```ts
// There are no props for this component.
```

## Defaults
```js
tag = 'h2' 
contrast = true
```

## Examples
### Color
```astro live
---
import { Title } from '@fulldevlabs/fullui'
---

<Title>The quick brown fox jumps over the lazy dog</Title=>   
<Title color="accent">The quick brown fox jumps over the lazy dog</Title=>  
```

### Contrast
```astro live
---
import { Title } from '@fulldevlabs/fullui'
---

<Title contrast={false}>The quick brown fox jumps over the lazy dog</Title=>   
<Title contrast={false} color="accent">The quick brown fox jumps over the lazy dog</Title=>  
```

### Size
```astro live
---
import { Title } from '@fulldevlabs/fullui'
--- 
<Title size="3">The quick brown fox jumps over the lazy dog</Title>   
<Title size="2">The quick brown fox jumps over the lazy dog</Title>  
<Title size="1">The quick brown fox jumps over the lazy dog</Title>   
```

### Compact
*It might seem weird having sizes and also compact, but it has a lot of benefits. More on this later*
```astro live
---
import { Title } from '@fulldevlabs/fullui'
--- 
<Title size="3" compact>The quick brown fox jumps over the lazy dog</Title>   
<Title size="2" compact>The quick brown fox jumps over the lazy dog</Title>  
<Title size="1" compact>The quick brown fox jumps over the lazy dog</Title>   
```
