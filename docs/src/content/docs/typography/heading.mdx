A heading element.

<br/>

```astro live
---
import { Heading } from '@fullui/ui'
---

<Heading>The quick brown fox jumps</Heading>
```

## Props
### `color`
**Type:** `color?: 'base' | 'lowered'`
```astro live
---
import { Heading } from '@fullui/ui'
---

<Heading color="base">The quick brown fox jumps</Heading>
<Heading color="lowered">The quick brown fox jumps</Heading>
```

### `size`
**Type:** `size?: 'xs' | 'sm' | 'md' | 'lg' | 'xl' | '2xl'`
```astro live
---
import { Heading } from '@fullui/ui'
---

<Heading size="xs">The quick brown fox jumps</Heading>
<Heading size="sm">The quick brown fox jumps</Heading>
<Heading size="md">The quick brown fox jumps</Heading>
<Heading size="lg">The quick brown fox jumps</Heading>
<Heading size="xl">The quick brown fox jumps</Heading>
<Heading size="2xl">The quick brown fox jumps</Heading>
```

### `density`
**Type:** `density?: 'base' | 'compact'`

Changes the underlying values of size.

This allows us to use the same components in different contexts, like in cards.

```astro live
---
import { Heading } from '@fullui/ui'
---

<Heading density="compact" size="sm">The quick brown fox jumps</Heading>
```
