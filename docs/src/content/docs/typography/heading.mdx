Astro heading component for rendering h1-h6 tags.

### Props
extends [Tag](/utility/tag) and [Children](/utility/children).

### Defaults
```ts
as = 'h2' 
contrast = true
```

## Examples
The examples below use regular syntax, check out [Syntax](/overview/syntax) for more options.

### Color
```astro live
---
import { Heading } from '@fullui/ui'
---

<Heading>The quick brown fox jumps over the lazy dog</Heading=>   
<Heading color="accent">The quick brown fox jumps over the lazy dog</Heading=>  
```

### Contrast
```astro live
---
import { Heading } from '@fullui/ui'
---

<Heading contrast={false}>The quick brown fox jumps over the lazy dog</Heading=>   
<Heading contrast={false} color="accent">The quick brown fox jumps over the lazy dog</Heading=>  
```

### Size
```astro live
---
import { Heading } from '@fullui/ui'
--- 
<Heading size="large">The quick brown fox jumps over the lazy dog</Heading>   
<Heading size="medium">The quick brown fox jumps over the lazy dog</Heading>  
<Heading size="small">The quick brown fox jumps over the lazy dog</Heading>   
```

### Compact
*It might seem weird having sizes and also compact, but it has a lot of benefits. More on this later*
```astro live
---
import { Heading } from '@fullui/ui'
--- 
<Heading size="large" compact>The quick brown fox jumps over the lazy dog</Heading>   
<Heading size="medium" compact>The quick brown fox jumps over the lazy dog</Heading>  
<Heading size="small" compact>The quick brown fox jumps over the lazy dog</Heading>   
```
