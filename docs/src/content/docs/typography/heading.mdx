Astro heading component for rendering h1-h6 tags.

## Props
extends [Tag](/components/tag) and [Children](/components/children).
```ts
// There are no props for this component.
```

## Defaults
```js
as = 'h2' 
contrast = true
```

## Examples
### Color
```astro live
---
import { Heading } from '@fullui/ui'
---

<Heading>The quick brown fox jumps over the lazy dog</Heading=>   
<Heading color="accent">The quick brown fox jumps over the lazy dog</Heading=>  
```

### Contrast
```astro live
---
import { Heading } from '@fullui/ui'
---

<Heading contrast={false}>The quick brown fox jumps over the lazy dog</Heading=>   
<Heading contrast={false} color="accent">The quick brown fox jumps over the lazy dog</Heading=>  
```

### Size
```astro live
---
import { Heading } from '@fullui/ui'
--- 
<Heading size="3">The quick brown fox jumps over the lazy dog</Heading>   
<Heading size="2">The quick brown fox jumps over the lazy dog</Heading>  
<Heading size="1">The quick brown fox jumps over the lazy dog</Heading>   
```

### Compact
*It might seem weird having sizes and also compact, but it has a lot of benefits. More on this later*
```astro live
---
import { Heading } from '@fullui/ui'
--- 
<Heading size="3" compact>The quick brown fox jumps over the lazy dog</Heading>   
<Heading size="2" compact>The quick brown fox jumps over the lazy dog</Heading>  
<Heading size="1" compact>The quick brown fox jumps over the lazy dog</Heading>   
```
