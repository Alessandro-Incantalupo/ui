## Props
Below props are available in addition to all [Tag](/utility/tag) and [Children](/utility/children) props.

#### `variant`
**Type:** `variant?: 'base' | 'lowered'`
```astro live
---
import { Tagline } from '@fullui/ui'
---

<Tagline variant="base">Lorem ipsum</Tagline>
<Tagline variant="lowered">Lorem ipsum</Tagline>
```

#### `size`
**Type:** `size?:  'sm' | 'md' | 'lg'`
```astro live
---
import { Tagline } from '@fullui/ui'
---

<Tagline size="sm">Lorem ipsum</Tagline>
<Tagline size="md">Lorem ipsum</Tagline>
<Tagline size="lg">Lorem ipsum</Tagline>
```

#### `compact`
**Type:** `compact?: boolean`

In compact mode size `xs` is equal too `sm` and size `xl` / `2xl` are equal to `lg`
```astro live
---
import { Tagline } from '@fullui/ui'
---

<Tagline compact size="sm">Lorem ipsum again</Tagline>
<Tagline compact size="md">Lorem ipsum again</Tagline>
<Tagline compact size="lg">Lorem ipsum again</Tagline>
```
