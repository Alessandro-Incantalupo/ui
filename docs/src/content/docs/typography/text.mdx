---
title: Text
description: Astro text component for a variety of text tags, like 'p' and more
---

### Props
extends [Tag](/components/tag) and [Children](/components/children).

### Defaults
```ts
as = 'p' 
```

## Examples
### Color
```astro live
---
import { Text } from '@fullui/ui'
---

<Text>The quick brown fox jumps over the lazy dog</Text=>   
<Text color="accent">The quick brown fox jumps over the lazy dog</Text=>  
```

### Contrast
```astro live
---
import { Text } from '@fullui/ui'
---

<Text contrast={true}>The quick brown fox jumps over the lazy dog</Text=>   
<Text contrast={true} color="accent">The quick brown fox jumps over the lazy dog</Text=>  
```

### Size
```astro live
---
import { Text } from '@fullui/ui'
--- 
<Text size="3">The quick brown fox jumps over the lazy dog</Text>   
<Text size="2">The quick brown fox jumps over the lazy dog</Text>  
<Text size="1">The quick brown fox jumps over the lazy dog</Text>   
```

### Compact
*It might seem weird having sizes and also compact, but it has a lot of benefits. More on this later*
```astro live
---
import { Text } from '@fullui/ui'
--- 
<Text size="3" compact>The quick brown fox jumps over the lazy dog</Text>   
<Text size="2" compact>The quick brown fox jumps over the lazy dog</Text>  
<Text size="1" compact>The quick brown fox jumps over the lazy dog</Text>   
```
