---
title: Text
description: Astro Text component for a variety of text tags, like 'p' and more.
---

### Props
Extends [Tag](/utility/tag), [Children](/utility/children).

### Defaults
```ts
as = 'p' 
```

## Examples
The examples below use regular syntax, check out [Syntax](/overview/syntax) for more options.

### Color
```astro live
---
import { Text } from '@fullui/ui'
---

<Text>The quick brown fox jumps over the lazy dog</Text=>   
<Text color="accent">The quick brown fox jumps over the lazy dog</Text=>  
```

### Contrast
```astro live
---
import { Text } from '@fullui/ui'
---

<Text contrast={true}>The quick brown fox jumps over the lazy dog</Text=>   
<Text contrast={true} color="accent">The quick brown fox jumps over the lazy dog</Text=>  
```

### Size
```astro live
---
import { Text } from '@fullui/ui'
--- 
<Text size="lg">The quick brown fox jumps over the lazy dog</Text>   
<Text size="md">The quick brown fox jumps over the lazy dog</Text>  
<Text size="sm">The quick brown fox jumps over the lazy dog</Text>   
```

### Compact
Why size and compact? Read [the page about sizes](/theme/size).
```astro live
---
import { Text } from '@fullui/ui'
--- 
<Text size="lg" compact>The quick brown fox jumps over the lazy dog</Text>   
<Text size="md" compact>The quick brown fox jumps over the lazy dog</Text>  
<Text size="sm" compact>The quick brown fox jumps over the lazy dog</Text>   
```
