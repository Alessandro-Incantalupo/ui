---
title: Text
description: Astro Text component for a variety of text tags, like 'p' and more.
---

### Props
Extends [Tag](/utility/tag), [Children](/utility/children).

### Defaults
```ts
as = 'p' 
```

## Examples
The examples below use regular syntax, check out [Syntax](/overview/components) for more options.

### Color
```astro live
---
import { Paragraph } from '@fullui/ui'
---

<Paragraph>The quick brown fox jumps over the lazy dog</Text=>   
<Paragraph color="accent">The quick brown fox jumps over the lazy dog</Text=>  
```

### Contrast
```astro live
---
import { Paragraph } from '@fullui/ui'
---

<Paragraph contrast={true}>The quick brown fox jumps over the lazy dog</Text=>   
<Paragraph contrast={true} color="accent">The quick brown fox jumps over the lazy dog</Text=>  
```

### Size
```astro live
---
import { Paragraph } from '@fullui/ui'
--- 
<Paragraph size="lg">The quick brown fox jumps over the lazy dog</Text>   
<Paragraph size="md">The quick brown fox jumps over the lazy dog</Text>  
<Paragraph size="sm">The quick brown fox jumps over the lazy dog</Text>   
```

### Compact
Why size and compact? Read [the page about sizes](/theme/size).
```astro live
---
import { Paragraph } from '@fullui/ui'
--- 
<Paragraph size="lg" compact>The quick brown fox jumps over the lazy dog</Text>   
<Paragraph size="md" compact>The quick brown fox jumps over the lazy dog</Text>  
<Paragraph size="sm" compact>The quick brown fox jumps over the lazy dog</Text>   
```
