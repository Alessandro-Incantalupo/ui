Equal columns above a certain breakpoint, based on CSS grid.

```astro live
---
import { Heading, Image, Split } from 'fulldev-ui'
---

<Split>
  <Heading>Heading</Heading>
  <Image src="/example.jpg" />
</Split>
```

## Props

### `align`

**Type:** `'start' | 'center' | 'end' = 'start'`

```astro live
---
import { Heading, Image, Split } from 'fulldev-ui'
---

<Split align="center">
  <Heading>Heading</Heading>
  <Image src="/example.jpg" />
</Split>
```

```astro live
---
import { Heading, Image, Split } from 'fulldev-ui'
---

<Split align="end">
  <Heading>Heading</Heading>
  <Image src="/example.jpg" />
</Split>
```
