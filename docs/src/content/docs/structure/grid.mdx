Structures children (primarily `Card`'s) in a grid.

```astro live
---
import { Card, Grid } from 'fulldev-ui'
---

<Grid>
  <Card>Card</Card>
  <Card>Card</Card>
  <Card>Card</Card>
</Grid>
```

## Props

### `size`

<SizeProp />

This also defined the amount of columns, together with the viewport width.

```astro live
---
import { Card, Grid } from 'fulldev-ui'
---

<Grid size="sm">
  <Card>Card</Card>
  <Card>Card</Card>
  <Card>Card</Card>
  <Card>Card</Card>
</Grid>
```

```astro live
---
import { Card, Grid } from 'fulldev-ui'
---

<Grid size="md">
  <Card>Card</Card>
  <Card>Card</Card>
  <Card>Card</Card>
</Grid>
```

```astro live
---
import { Card, Grid } from 'fulldev-ui'
---

<Grid size="lg">
  <Card>Card</Card>
  <Card>Card</Card>
</Grid>
```
