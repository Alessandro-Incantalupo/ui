Structures children in a vertical stack and sets dynamic margins between them.

```astro live
---
import { Button, Heading, Paragraph, Stack } from 'fulldev-ui'
---

<Stack>
  <Heading>Heading</Heading>
  <Paragraph>Paragraph</Paragraph>
  <Button>Button</Button>
</Stack>
```

## Props

### `align`

**Type:** `'start' | 'center' | 'end' = 'start'`

```astro live
---
import { Button, Heading, Paragraph, Stack } from 'fulldev-ui'
---

<Stack align="center">
  <Heading>Heading</Heading>
  <Paragraph>Paragraph</Paragraph>
  <Button>Button</Button>
</Stack>
```

```astro live
---
import { Button, Heading, Paragraph, Stack } from 'fulldev-ui'
---

<Stack align="end">
  <Heading>Heading</Heading>
  <Paragraph>Paragraph</Paragraph>
  <Button>Button</Button>
</Stack>
```
