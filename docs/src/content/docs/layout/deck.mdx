Astro deck component for grouping cards together.

<br />

```astro live
---
import { Deck, DecorativeBox } from '@fullui/ui'
---

<Deck>
  <DecorativeBox />
  <DecorativeBox />
  <Deck /></Deck
>
```

## Props

### `html`

**Type:** `html?: string`

```astro live
---
import { Deck, DecorativeBox } from '@fullui/ui'
---

<Deck html="<i>Deck</i>">
  <DecorativeBox />
  <DecorativeBox />
  <DecorativeBox />
  <Deck /></Deck
>
```

### `layout`

**Type:** `layout?: 'carousel' | 'grid' | 'masonry' | 'row' | 'panels' | 'bento'`

```astro live
---
import { Deck, DecorativeBox } from '@fullui/ui'
---

<Deck layout="carousel">
  <DecorativeBox />
  <DecorativeBox />
  <DecorativeBox />
  <Deck /></Deck
>
```
