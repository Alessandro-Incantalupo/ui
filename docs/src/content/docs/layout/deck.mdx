Astro deck component for grouping cards together.

:::note
This component is currenty in beta.
If you find any problems or bugs report them here: [GitHub Issues](https://github.com/fulldevlabs/fullui/issues).
:::

## Props
extends [Tag](/components/tag) and [Children](/components/children).
```ts
layout?: 'carousel' | 'grid' | 'masonry' | 'row' | 'panels' | 'bento'
```

## Examples
```astro live
---
import { Deck } from '@fullui/ui'
import { DecorativeBox } from '@fullui/ui'
---

<Deck layout="carousel">
  <DecorativeBox width="100%" height="150px" />
  <DecorativeBox width="100%" height="150px" />
  <DecorativeBox width="100%" height="150px" />
  <DecorativeBox width="100%" height="150px" />
<Deck/>
```
