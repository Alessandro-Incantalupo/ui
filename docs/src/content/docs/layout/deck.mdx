Astro deck component for grouping cards together.

<br/>

```astro live
---
import { Deck, DecorativeBox } from '@fullui/ui'
---

<Deck>
  <DecorativeBox />
  <DecorativeBox />
<Deck/>
```

## Props
### `html`
**Type:** `html?: string`

```astro live
---
import { Deck, DecorativeBox } from '@fullui/ui'
---

<Deck html="<i>Deck</i>">
  <DecorativeBox />
  <DecorativeBox />
  <DecorativeBox />
<Deck/>
```
