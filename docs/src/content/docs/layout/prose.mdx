Astro prose component used for formatting your text.

## Example
The example below uses regular syntax, check out [Syntax](/overview/syntax) for more options.

```astro live
---
import { Prose, Subtext, Heading, Text } from '@fullui/ui'
---

<Prose align="start">
  <Heading>
    Lorem ipsum dolor
  </Heading>
  <Text>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, <br />
    sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
  </Text>
  <Subtext>
    Lorem ipsum
  </Subtext>
</Prose>
```
