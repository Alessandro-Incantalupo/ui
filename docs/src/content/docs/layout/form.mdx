Astro form component used for creating a form.

<br />

## Props

### `html`

**Type:** `html?: string`

```astro live
---
import { Button, Form, Input } from '@fullui/ui'
---

<Form html="<i>Form</i>">
  <Input
    name="password"
    placeholder="Password"
    type="password"
  />
  <Button
    type="submit"
    variant="surface"
    >Submit</Button
  >
</Form>
```
