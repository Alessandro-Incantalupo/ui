Displays details with a summary.

## Props
extends [Tag](/utility/tag) and [Children](/utility/children).
```ts
summary?: string
```

## Defaults
```ts
as = 'details'
```

## Examples
The example below uses regular syntax, check out [Syntax](/overview/components) for more options.

```astro live
---
import { Details, Summary } from '@fullui/ui'
---

<Details>
  <Summary>Lorem ipsum dolor</Summary>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit, <br />
  sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
</Details>
```
