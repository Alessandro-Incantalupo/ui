## Props
extends [Tag](/utility/tag), [Children](/utility/children).

```ts
direction?: 'column' | 'row'
frame?: 'screen' | 'panel'
variant?: 'solid' | 'soft' | 'ghost'
align?: 'start' | 'center'
collection?: string
slug?: string
```

## Defaults
```ts
as = 'section'
```
