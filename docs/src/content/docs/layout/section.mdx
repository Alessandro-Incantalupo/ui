A container used for full-width blocks which sets:

- Outer margins and paddings
- Layout inside the container
- Spacing between child elements

<br />

```astro live
---
import { Button, Heading, Paragraph, Section } from '@fullui/ui'
---

<Section>
  <Heading>Heading</Heading>
  <Heading>Heading</Heading>
  <Paragraph>Paragraph</Paragraph>
  <Button>Button</Button>
</Section>
```

## Props

### `direction`

**Type:** `direction?: 'vertical' | 'horizontal'`

```astro live
---
import { Button, Heading, Section } from '@fullui/ui'
---

<Section direction="vertical">
  <Heading>Heading</Heading>
  <Button>Button</Button>
</Section>
<Section direction="horizontal" layout="undefined">
  <Heading>Heading</Heading>
  <Button>Button</Button>
</Section>
```

### `x`

**Type:** `x?: 'start' | 'center' | 'end' | 'between' | 'stretch'`
**Type:** `x?: 'left' | 'center' | 'right' | 'spread' | 'fill'`
**Type:** `y?: 'top' | 'center' | 'bottom' | 'spread' | 'fill'`
{/* TODO kan ook voor deck! spreak is grid met ruimtes, fill is cover, top is masonry rechte line top, bottom bottom etc */}

```astro live
---
import { Button, Heading, Section } from '@fullui/ui'
---

<Section x="left">
  <Heading>Heading</Heading>
  <Button>Button</Button>
</Section>
<Section x="center">
  <Heading>Heading</Heading>
  <Button>Button</Button>
</Section>
<Section x="right">
  <Heading>Heading</Heading>
  <Button>Button</Button>
</Section>
<Section x="fill">
  <Heading>Heading</Heading>
  <Button>Button</Button>
</Section>
<Section direction="horizontal" x="spread">
  <Heading>Heading</Heading>
  <Button>Button</Button>
</Section>
```

### `y`

**Type:** `y?: 'start' | 'center' | 'end' | 'between' | 'stretch'`

```astro live
---
import { DecorativeBox, Section } from '@fullui/ui'
---

<Section direction="horizontal" y="bottom">
  <DecorativeBox />
  <Section /></Section
>
```

### `density`

**Type:** `density?: 'base' | 'compact'`

Changes the underlying values of size.

This allows us to use the same components in different contexts, like in cards.

```astro live
---
import { DecorativeBox, Section } from '@fullui/ui'
---

<Section direction="horizontal" density="compact" y="bottom">
  <DecorativeBox />
  <DecorativeBox height="200px" />
  <Section /></Section
>
```
