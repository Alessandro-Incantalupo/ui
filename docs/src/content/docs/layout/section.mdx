A section element.

<br/>

```astro live
---
import { Section, DecorativeBox } from '@fullui/ui'
---

<Section>
  <DecorativeBox />
<Section/>
```

## Props

### `layout`
**Type:** `layout?: 'column' | 'row' | 'split'`

### `x`
**Type:** `x?: 'start' | 'center' | 'end' | 'between' | 'stretch'`

### `y`
**Type:** `y?: 'start' | 'center' | 'end' | 'between' | 'stretch'`

## Defaults
```ts
as = 'section'
```

## Examples

```astro live
---
import { Section, DecorativeBox } from '@fullui/ui'
---

<Section>
  <DecorativeBox />
  <DecorativeBox height="200px" />
<Section/>
```

### Layout
```astro live
---
import { Section, DecorativeBox } from '@fullui/ui'
---

<Section layout="split">
  <DecorativeBox />
  <DecorativeBox height="200px" />
<Section/>
```

### Alignment
```astro live
---
import { Section, DecorativeBox } from '@fullui/ui'
---

<Section layout="row" y="end">
  <DecorativeBox />
  <DecorativeBox height="200px" />
<Section/>
```
