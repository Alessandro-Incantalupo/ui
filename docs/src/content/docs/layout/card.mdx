Astro card component used displaying a card.

## Props
extends [Tag](/utility/tag) and [Children](/utility/children).
```ts
frame?: 'fill' | 'panel' | 'screen'
variant?: 'ghost' | 'soft' | 'solid'
```

## Examples
The examples below use regular syntax, check out [Syntax](/overview/syntax) for more options.

```astro live
---
import { Card, Image, Writeup, Heading, Text } from '@fullui/ui'
---

<Card>
  <Image src="https://picsum.photos/id/237/336/154" />
  <Writeup>
    <Heading>
      Lorem ipsum dolor
    </Heading>
    <Text>
      Lorem ipsum dolor sit amet, consectetur adipiscing elit, <br />
      sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
    </Text>
  </Writeup>
</Card>
```
