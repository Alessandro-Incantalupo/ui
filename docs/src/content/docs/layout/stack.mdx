Astro stack component used for alignment and direction.

:::note
This component is currenty in beta.
If you find any problems or bugs report them here: [GitHub Issues](https://github.com/fulldevlabs/fullui/issues).
:::

## Props
extends [Tag](/components/tag) and [Children](/components/children).
```ts
direction: 'column' | 'row' | null | undefined
align: 'start' | 'center' | null | undefined
```

## Examples
The examples down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).

```astro live
---
import { Stack } from '@fullui/ui'
import { DecorativeBox } from '@fullui/ui'
---

<Stack align="center" direction="column">
  <DecorativeBox width="45%" height="300px" />
  <DecorativeBox width="45%" height="300px" />
</Stack>
```

```astro live
---
import { Stack } from '@fullui/ui'
import { DecorativeBox } from '@fullui/ui'
---

<Stack align="start" direction="row">
  <DecorativeBox width="45%" height="300px" />
  <DecorativeBox width="45%" height="300px" />
</Stack>
```
