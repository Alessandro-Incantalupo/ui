A layout component that stacks children vertically or horizontally.

<br/>

```astro live
---
import { Stack, DecorativeBox } from '@fullui/ui'
---

<Stack>
  <DecorativeBox />
  <DecorativeBox height="200px" />
<Stack/>
```

## Props

### `layout`
**Type:** `layout?: 'column' | 'row' | 'split'`

### `x`
**Type:** `x?: 'start' | 'center' | 'end' | 'between' | 'stretch'`

### `y`
**Type:** `y?: 'start' | 'center' | 'end' | 'between' | 'stretch'`

### `density`
**Type:** `density?: 'base' | 'compact'`

Changes the underlying values of size.

This allows us to use the same components in different contexts, like in cards.

### `theme`
<Theming/>

```astro live
---
import { Stack, DecorativeBox } from '@fullui/ui'
---

<Stack layout="row" density="compact" y="end">
  <DecorativeBox />
  <DecorativeBox height="200px" />
<Stack/>
```
