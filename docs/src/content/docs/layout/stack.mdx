## Props
extends [Tag](/foundation/tag), [Children](/foundation/children).

```ts
layout?: 'column' | 'row' | 'split'
x?: 'start' | 'center' | 'end' | 'between' | 'stretch'
y?: 'start' | 'center' | 'end' | 'between' | 'stretch'
```

## Defaults
```ts
as = 'section'
```
