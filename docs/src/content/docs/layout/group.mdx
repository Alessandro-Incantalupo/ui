Astro group component used for grouping other components together.

## Props
extends [Tag](/utility/tag) and [Children](/utility/children).
```ts
layout?: 'column' | 'row'
align?: 'start' | 'center'
```

## Examples
Here are some examples of ways you can use this component.

### Default
```astro live
---
import { Group } from '@fullui/ui'
import { DecorativeBox } from '@fullui/ui'
---

<Group>
  <DecorativeBox width="205px" height="150px" />
  <DecorativeBox width="205px" height="150px" />
  <DecorativeBox width="205px" height="150px" />
<Group/>
```
<br />

### Different heights
Other components will change height to match the tallest unless the height is specified.
```astro live
---
import { Group } from '@fullui/ui'
import { DecorativeBox } from '@fullui/ui'
---

<Group layout="column">
  <DecorativeBox width="100%" height="150px" />
  <DecorativeBox width="205px" height="200px" />
  <DecorativeBox width="205px" height="100px" />
  <DecorativeBox width="205px" />
<Group/>
```
