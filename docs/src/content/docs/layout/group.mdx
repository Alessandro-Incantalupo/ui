A layout component that groups children vertically or horizontally.

<br />

```astro live
---
import { DecorativeBox, Group } from '@fullui/ui'
---

<Group>
  <DecorativeBox />
  <DecorativeBox height="200px" />
</Group>
```

## Props

### `layout`

**Type:** `layout?: 'column' | 'row' | 'split'`

```astro live
---
import { DecorativeBox, Group } from '@fullui/ui'
---

<Group direction="horizontal">
  <DecorativeBox />
  <DecorativeBox height="200px" />
</Group>
```

### `x`

**Type:** `x?: 'start' | 'center' | 'end' | 'between' | 'stretch'`

```astro live
---
import { DecorativeBox, Group } from '@fullui/ui'
---

<Group direction="horizontal" x="right">
  <DecorativeBox />
  <DecorativeBox height="200px" />
</Group>
```

### `y`

**Type:** `y?: 'start' | 'center' | 'end' | 'between' | 'stretch'`

```astro live
---
import { DecorativeBox, Group } from '@fullui/ui'
---

<Group direction="horizontal" y="bottom">
  <DecorativeBox />
  <DecorativeBox height="200px" />
</Group>
```

### `density`

**Type:** `density?: 'base' | 'compact'`

Changes the underlying values of size.

This allows us to use the same components in different contexts, like in cards.

```astro live
---
import { DecorativeBox, Group } from '@fullui/ui'
---

<Group direction="horizontal" density="compact" y="bottom">
  <DecorativeBox />
  <DecorativeBox height="200px" />
</Group>
```
