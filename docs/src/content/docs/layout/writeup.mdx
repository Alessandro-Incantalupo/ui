Astro writeup component used for laying out your text.

:::note
This component is currenty in beta.
If you find any problems or bugs report them here: [GitHub Issues](https://github.com/fullui/ui/issues).
:::

## Props
extends [Tag](/utility/tag) and [Children](/utility/children).
```ts
align?: 'start' | 'center'
```

## Example
The example down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).

```astro live
---
import { Writeup } from '@fullui/ui'
import { Subtext } from '@fullui/ui'
import { Heading } from '@fullui/ui'
import { Text } from '@fullui/ui'
import { Group } from '@fullui/ui'
import { Button } from '@fullui/ui'
---

<Writeup align="start">
  <Subtext>
    Lorem ipsum
  </Subtext>
  <Heading>
    Lorem ipsum dolor
  </Heading>
  <Text>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, <br />
    sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
  </Text>
  <Group>
    <Button variant="outline">
      Button
    </Button>
    <Button variant="soft">
      Button
    </Button>
  </Group>
</Writeup>
```
