Component similar to prose (insipred by [Tailwind](https://github.com/tailwindlabs/tailwindcss-typography)), but for smaller section or button contentConfigParser, rather that full articles.
Astro writeup component used for laying out your text.


## Props
Extends [Tag](/utility/tag), [Children](/utility/children).
```ts
align?: 'start' | 'center'
```

## Examples
The examples below could also be written with [different syntax](/overview/components).

```astro live
---
import { Writeup, Subtext, Heading, Text, Group, Button } from '@fullui/ui'
---

<Writeup align="start">
  <Subtext>
    Lorem ipsum
  </Subtext>
  <Heading>
    Lorem ipsum dolor
  </Heading>
  <Text>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, <br />
    sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
  </Text>
  <Group>
    <Button variant="surface">
      Button
    </Button>
    <Button variant="ghost">
      Button
    </Button>
  </Group>
</Writeup>
```
