You are able to print out a predefined array of buttons with the `ButtonGroup` components.

## Props
extends [Tag](/components/tag) and [Link](/components/link).
```ts
links: array | null | undefined
```

## Example
```astro live
---
import { LinkGroup } from '@fulldevlabs/fullui'

const myLinks = [
  {html: "First link"},
  {html: "Second link"},
  {html: "Third link"},
]
---

<LinkGroup links={myLinks} />
```

You are also able to specify a `color`, `contrast`, `size` and an `icon`.
```astro live
---
import { LinkGroup } from '@fulldevlabs/fullui'

const myLinks = [
  {html: "First link", size: "3", icon: {name: "arrow-right"}},
  {html: "Second link", size: "2", contrast: true},
  {html: "Third link", size: "1", color: "accent"},
]
---

<LinkGroup links={myLinks} />
```
