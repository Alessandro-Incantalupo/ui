Create an multiple input fields for use in a form.

## Props
extends [Tag](/components/tag), [Field](/components/field) and [Button](/components/button).
```ts
fields: array | null | undefined
```

## Example
```astro live
---
import { FieldGroup } from '@fulldevlabs/fullui'

const myFields = [
  {label: {html: "Email"}, input: {name: "email"}},
  {label: {html: "Subject"}, input: {name: "subject"}},
  {label: {html: "Message"}, textarea: {name: "message"}},
]
---

<FieldGroup fields={myFields} />
```
