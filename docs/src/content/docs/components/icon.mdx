Displays a button or a component that looks like a button.
## Props
Extends [Tag](/components/tag),
```ts
icon?: TablerIconName | null // The name of the Tabler icon
```

## Examples
Below examples use regular syntax, but work with [all syntaxes](/guides/components#syntax).

### Color
```astro live
---
import { Icon } from '@fulldevlabs/fullui'
---

<Icon icon="circle" />
<Icon icon="circle" color="accent" />
```

### Size
```astro live
---
import { Icon } from '@fulldevlabs/fullui'
---

<Icon icon="circle" size="large" />
<Icon icon="circle" size="medium" />
<Icon icon="circle" size="small" />
```

### Compact
*It might seem weird having sizes and also compact, but it has a lot of benefits. More on this later*
```astro live
---
import { Icon } from '@fulldevlabs/fullui'
---

<Icon icon="circle" size="large" compact />
<Icon icon="circle" size="medium" compact />
<Icon icon="circle" size="small" compact />
```
