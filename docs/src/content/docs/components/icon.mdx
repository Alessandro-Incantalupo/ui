## Props
Extends [Endomorphic](/guides/macromorphic), [Tag](/components/tag)
```ts
interface Props
  extends Endomorphic<Props, 'icon', keyof typeof tabler.icons>,
    ComponentProps<typeof Tag> {}
```

## Defaults
```ts
const { as = 'i' } = Astro.props
```

## Examples
Below examples use regular syntax, but work with [all syntaxes](/guides/components#syntax).

### Color
```astro live
---
import { Icon } from '@fulldevlabs/fullui'
---

<Icon icon="friends" />
<Icon icon="friends" color="accent" />
```

### Size
```astro live
---
import { Icon } from '@fulldevlabs/fullui'
---

<Icon icon="friends" size="4" />
<Icon icon="friends" size="3" />
<Icon icon="friends" size="2" />
<Icon icon="friends" size="1" />
```

### Compact
*It might seem weird having sizes and also compact, but it has a lot of benefits. More on this later*
```astro live
---
import { Icon } from '@fulldevlabs/fullui'
---

<Icon icon="friends" size="4" compact />
<Icon icon="friends" size="3" compact />
<Icon icon="friends" size="2" compact />
<Icon icon="friends" size="1" compact />
```
