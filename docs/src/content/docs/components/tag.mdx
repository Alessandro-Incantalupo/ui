This component is kind of a helper, almost every component in Fullui makes use of this component. You might find yourself explicitly using it, yet it's good to know how it works.

:::note
This component is currenty in beta.
If you find any problems or bugs report them here: [GitHub Issues](https://github.com/fulldevlabs/fullui/issues).
:::
