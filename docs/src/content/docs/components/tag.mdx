This component is kind of a helper, allmost every component in Fullui uses this component. You might find yourself explicitly using it, yet it's good to know how it works.

## Props
```ts
interface Props {
  as?: HTMLTag
  class?: HTMLAttributes<HTMLTag>['class']
  id?: HTMLAttributes<HTMLTag>['id']
  style?: HTMLAttributes<HTMLTag>['style']
  compact?: boolean | null
  scheme?: 'light' | 'dark' | null
  color?: 'base' | 'accent' | null
  size?: '1' | '2' | '3' | '4' | null
  variant?: 'solid' | 'soft' | 'surface' | 'outline' | 'ghost' | null
  contrast?: boolean | null
}
```
