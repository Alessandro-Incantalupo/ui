A logo that always redirects to the home page.

## Props
extends [Endomorphic](/guides/components), [Tag](/components/tag) and [Image](/components/image).
```ts
interface Props
  extends Endomorphic<'logo', Props, ComponentProps<typeof Image>['image']>,
    ComponentProps<typeof Tag>,
    ComponentProps<typeof Image> {}
```

## Defaults
```ts
const { as = 'a' } = Astro.props
```

## Example
The example down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).

```astro live
---
import { Logo } from '@fulldevlabs/fullui'
---

<Logo src="/./favicon.png" />
```
