## Props
extends [Endomorphic](/guides/components) and [Tag](/components/tag).
```ts
interface Props
  extends Endomorphic<'writeup', Props, string>,
    ComponentProps<typeof Tag>,
    Pick<ComponentProps<typeof Tagline>, 'tagline' | '_tagline'>,
    Pick<ComponentProps<typeof Title>, 'title' | '_title'>,
    Pick<ComponentProps<typeof Description>, 'description' | '_description'> {
  align?: 'start' | 'center' | null
}
```

## Example
The example down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).

```astro live
---
import { Writeup } from '@fulldevlabs/fullui'
---

<Writeup 
  tagline="This is a writeup" 
  title="What are writeups" 
  description="Lorem ipsum dolor sit amet, <br/> consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et."
/>
```
