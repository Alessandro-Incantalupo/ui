## Props
extends [Tag](/components/tag).
```ts
align: 'start' | 'center' | null | undefined
```

## Example
The example down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).

```astro live
---
import { Writeup } from '@fulldevlabs/fullui'

---

<Writeup
  tagline={{html: "This is a writeup"}}
  title={{html: "This is a writeup"}}
  description={{
    html: "Lorem ipsum dolor sit amet, <br/> consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et."
  }}
/>
```
