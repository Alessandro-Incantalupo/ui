Displays a link or a component that looks like a link.
## Props
Extends [Tag](/components/tag), [Icon](/components/icon),
```ts
link?: string | null // Text in the link
```

## Examples
Below examples use regular syntax, but work with [all syntaxes](/guides/components#syntax).

### Color
```astro live
---
import { Link } from '@fulldevlabs/fullui'
---

<Link href="#" look="solid">Link</Link>
<Link href="#" color="accent" look="soft">Link</Link>
```

### Contrast
```astro live
---
import { Link } from '@fulldevlabs/fullui'
---

<Link href="#" color="base" contrast>Link</Link>
<Link href="#" color="accent" contrast>Link</Link>
```

### Icon
```astro live
---
import { Link } from '@fulldevlabs/fullui'
---

<Link href="#" icon="arrow-right">Link</Link>
```

### Size
```astro live
---
import { Link } from '@fulldevlabs/fullui'
---

<Link href="#" size="large">Link</Link>
<Link href="#" size="medium">Link</Link>
<Link href="#" size="small">Link</Link>
```

### Compact
*It might seem weird having sizes and also compact, but it has a lot of benefits. More on this later*
```astro live
---
import { Link } from '@fulldevlabs/fullui'
---

<Link href="#" size="large" compact>Link</Link>
<Link href="#" size="medium" compact>Link</Link>
<Link href="#" size="small" compact>Link</Link>
```
