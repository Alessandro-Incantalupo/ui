---
title: Link
description: Displays a button or a component that looks like a button.
sidebar:
  order: 6
---

## Props
extends [Tag](/components/tag) and [Children](/components/children).
```ts
href: string | null | undefined
```

## Defaults
```ts
tag = 'a'
```

## Examples
Below examples use regular props, but also accept [Endomorphic props](/guides/macromorphic).

### Color
```astro live
---
import { Link } from '@fulldevlabs/fullui'
---
<Link href="#">Link</Link>
<Link href="#" color="accent">Link</Link>
```

### Contrast
```astro live
---
import { Link } from '@fulldevlabs/fullui'
---

<Link href="#" color="base" contrast>Link</Link>
<Link href="#" color="accent" contrast>Link</Link>
```

### Icon
```astro live
---
import { Link } from '@fulldevlabs/fullui'
---

<Link href="#" icon={{name: "arrow-right"}}>Link</Link>
```

### Size
```astro live
---
import { Link } from '@fulldevlabs/fullui'
---
<Link href="#" size="3">Link</Link>
<Link href="#" size="2">Link</Link>
<Link href="#" size="1">Link</Link>
```

### Compact
*It might seem weird having sizes and also compact, but it has a lot of benefits. More on this later*
```astro live
---
import { Link } from '@fulldevlabs/fullui'
---

<Link href="#" size="3" compact>Link</Link>
<Link href="#" size="2" compact>Link</Link>
<Link href="#" size="1" compact>Link</Link>
```
