## Props
extends [Tag](/components/tag), [Display](/components/display) and [Action](/components/action).
```ts
direction: 'column' | 'row' | null | undefined
align: 'start' | 'center' | null | undefined
```

## Examples
The examples down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).

```astro live
---
import { Composite } from '@fulldevlabs/fullui'
---

<Composite
  direction="row"
  display={{
    rating: {value: 5},
    writeup: {
      tagline: {html: "This is a composite"},
      title: {html: "This is a composite"},
      description: {
        html: "Lorem ipsum dolor sit amet, <br/> consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et."
      },
    }
  }}
/>
```
