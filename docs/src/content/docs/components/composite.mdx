## Props
extends [Endomorphic](/guides/components), [Tag](/components/tag), [Display](/components/display) and [Action](/components/action)
```ts
interface Props
  extends Endomorphic<'composite', Props>,
    ComponentProps<typeof Tag>,
    ComponentProps<typeof Display>,
    ComponentProps<typeof Action> {
  direction?: 'column' | 'row' | null
  align?: 'start' | 'center' | null
}
```

## Examples
The examples down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).

```astro live
---
import { Composite } from '@fulldevlabs/fullui'
import { Title } from '@fulldevlabs/fullui'

const myButtons = [
  {"button": "Primary Button"},
  {"button": "Secondary Button"},
]
---
{/* TODO de <br> staat op een plek die niet mooi is voor het design, de tweede line is nu langer dan de eerste wat er niet mooi uitzien. Display heb ik al aangepast, dus zie dat als voorbeeld. */}
<Composite
  tagline="This is a composite",
  title="Composite!",
  description="Lorem ipsum dolor sit amet, <br/> consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et."
  buttons={myButtons}
  align="center" 
  direction="column" 
/>
```

### Direction
You are able to choose the direction of your composite.

```astro live
---
import { Composite } from '@fulldevlabs/fullui'
import { Title } from '@fulldevlabs/fullui'
---

<Composite
  writeup:
    title="Composite!",
    description="Lorem ipsum dolor sit amet, <br/> consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et."
  rating=5
  direction="column" 
/>
```
