## Props
extends [BuildProps](/guides/components), [Tag](/components/tag), [Button](/components/button)
```ts
interface Props
  extends BuildProps<'composite', Props>,
    ComponentProps<typeof Tag>,
    ComponentProps<typeof Display>,
    ComponentProps<typeof Action> {
  direction?: 'column' | 'row' | null
  align?: 'start' | 'center' | null
}
```

## Examples
The examples down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).

```astro live
---
import { Composite } from '@fulldevlabs/fullui'
import { Title } from '@fulldevlabs/fullui'
---

<Composite align="center" direction="column">
  <Title>I'm a composite!</Title>
</Composite>
```

### Alignment
You are able to align composites the way you want.

```astro live
---
import { Composite } from '@fulldevlabs/fullui'
import { Title } from '@fulldevlabs/fullui'
---

<Composite align="start">
  <Title>I'm a default composite!</Title>
</Composite>
```

```astro live
---
import { Composite } from '@fulldevlabs/fullui'
import { Title } from '@fulldevlabs/fullui'
---

<Composite align="center">
  <Title>I'm a centered composite!</Title>
</Composite>
```

### Direction
You are able to choose the direction of your composite.

```astro live
---
import { Composite } from '@fulldevlabs/fullui'
import { Title } from '@fulldevlabs/fullui'
---

<Composite direction="column">
  <Title>I'm a composit in columns!</Title>
</Composite>
```

```astro live
---
import { Composite } from '@fulldevlabs/fullui'
import { Title } from '@fulldevlabs/fullui'
---

<Composite direction="row">
  <Title>I'm a composit in rows!</Title>
</Composite>
```
