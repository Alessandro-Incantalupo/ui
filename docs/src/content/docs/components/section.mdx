## Props
extends [Endomorphic](/guides/components), [Tag](/components/tag), [Composite](/components/composite), [Image](/components/image), [Prose](/components/prose), [Cards](/components/cards) and [Icon](/components/icon).
```ts
interface Props
  extends Endomorphic<'section', Props>,
    ComponentProps<typeof Tag>,
    ComponentProps<typeof Composite>,
    ComponentProps<typeof Image>,
    Pick<ComponentProps<typeof Prose>, 'prose' | '_prose'>,
    Pick<ComponentProps<typeof Cards>, 'cards' | '_cards'>,
    Pick<ComponentProps<typeof Icon>, 'icon' | '_icon'> {
  direction?: 'row' | 'column' | null
  align?: 'start' | 'center' | null
  frame?: 'solid' | 'soft' | 'ghost' | null
}
```

## Example
The example down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).

```astro live
---
import { Section } from '@fulldevlabs/fullui'

const myButtons = [
  {"button": "Primary Button"},
  {"button": "Secondary Button"},
]
---
<Section 
  tagline="This is a composite",
  title="Composite!",
  description="Lorem ipsum dolor sit amet, <br/> consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et."
  buttons={myButtons}
  align="center"
/>
```
