## Props
extends [BuildProps](/guides/components), [Tag](/components/tag), [Composite](/components/composite), [Image](/components/image), [Prose](/components/prose), [Cards](/components/cards), [Icon](/components/icon)
```ts
interface Props
  extends BuildProps<'section', Props>,
    ComponentProps<typeof Tag>,
    ComponentProps<typeof Composite>,
    ComponentProps<typeof Image>,
    Pick<ComponentProps<typeof Prose>, 'prose' | '_prose'>,
    Pick<ComponentProps<typeof Cards>, 'cards' | '_cards'>,
    Pick<ComponentProps<typeof Icon>, 'icon' | '_icon'> {
  direction?: 'row' | 'column' | null
  align?: 'start' | 'center' | null
  frame?: 'solid' | 'soft' | 'ghost' | null
}
```
