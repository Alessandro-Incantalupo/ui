A component for your ratings from 1 to 5.

:::caution
This component is currenty work in progress.
If you find any problems or bugs report them here: [GitHub Issues](https://full.dev).
:::

## Props
extends [Endomorphic](/guides/components) and [Tag](/components/tag).
```ts
interface Props
  extends Endomorphic<Props, 'rating', number>,
    ComponentProps<typeof Tag> {
  align?: 'start' | 'center' | null
}
```

The example down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).
