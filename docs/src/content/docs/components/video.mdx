## Example
The example down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).

:::note
This component only accepts `youtube.com` and `youtu.be` urls.
:::

```astro live
---
import { Video } from '@fulldevlabs/fullui'
---

<Video video="https://youtu.be/dQw4w9WgXcQ?si=SSmA4Z0_wrwor6Xg" />
```
