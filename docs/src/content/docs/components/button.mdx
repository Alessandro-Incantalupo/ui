Displays a button or a component that looks like a button.
## Props
Extends [Tag](/components/tag), [Icon](/components/icon),
```ts
button?: string | null // Text in the button
```

## Examples
Below examples use regular syntax, but work with [all syntaxes](/guides/components#syntax).

### Look
```astro live
---
import { Button } from '@fulldevlabs/fullui'
---

<Button look="solid">Button</Button>
<Button look="soft">Button</Button>
<Button look="ghost">Button</Button>
```

### Color
```astro live
---
import { Button } from '@fulldevlabs/fullui'
---

<Button color="base" look="solid">Button</Button>
<Button color="base" look="soft">Button</Button>
<Button color="base" look="ghost">Button</Button>
```

### Contrast
```astro live
---
import { Button } from '@fulldevlabs/fullui'
---

<Button color="base" look="solid" contrast>Button</Button>
<Button color="base" look="soft" contrast>Button</Button>
<Button color="base" look="ghost" contrast>Button</Button>
```

### Icon
```astro live
---
import { Button } from '@fulldevlabs/fullui'
---

<Button icon="arrow-right" />
<Button icon="arrow-right">Button</Button>
```

### Size
```astro live
---
import { Button } from '@fulldevlabs/fullui'
---

<Button size="large">Button</Button>
<Button size="medium">Button</Button>
<Button size="small">Button</Button>
```

### Compact
*It might seem weird having sizes and also compact, but it has a lot of benefits. More on this later*
```astro live
---
import { Button } from '@fulldevlabs/fullui'
---

<Button size="large" compact>Button</Button>
<Button size="medium" compact>Button</Button>
<Button size="small" compact>Button</Button>
```
