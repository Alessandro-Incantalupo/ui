## Props
extends [BuildProps](/guides/components), [Tag](/components/tag) and [Icon](/components/icon)
```ts
interface Props
  extends BuildProps<'button', Props, string>,
    ComponentProps<typeof Tag>,
    Pick<ComponentProps<typeof Icon>, 'icon' | '_icon'> {
  look?: 'solid' | 'soft' | 'ghost' | null
  contrast?: boolean | null
}
```

## Examples
Below examples use regular syntax, but this component works with [all syntaxes](/guides/components#syntax).

### Look
```astro live
---
import { Button } from '@fulldevlabs/fullui'
---

<Button look="solid">Button</Button>
<Button look="soft">Button</Button>
<Button look="ghost">Button</Button>
```

### Color
```astro live
---
import { Button } from '@fulldevlabs/fullui'
---

<Button color="base" look="solid">Button</Button>
<Button color="base" look="soft">Button</Button>
<Button color="base" look="ghost">Button</Button>
```

### Contrast
```astro live
---
import { Button } from '@fulldevlabs/fullui'

<Button color="base" look="soft" contrast>Button</Button>
<Button color="base" look="ghost" contrast>Button</Button>
```

