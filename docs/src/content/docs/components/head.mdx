A simple head element for your metadata and SEO needs.

## Props
```ts
title: string | null | undefined
description: string | null | undefined
image: string | null | undefined
```

## Example

```astro
---
import { Head } from '@fulldevlabs/fullui'
---

<Head 
  title="Fullui" 
  description="Fullui is component library and design system for Astro with powerfull features that you probably have not seen in other libraries before." 
  image="..."
/>
```
