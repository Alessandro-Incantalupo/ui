## Props
extends [Tag](/components/tag).
```ts
src: string | null | undefined
alt: string | null | undefined
loading: string | null | undefined
mask: boolean | null | undefined
sizes: string | null | undefined
position: string | null | undefined
frame: string | null | undefined
```

## Defaults
```ts
tag = 'img'
```

## Example
The example down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).

```astro live
---
import { Image } from '@fulldevlabs/fullui'
---

<Image 
  src="https://picsum.photos/id/237/536/354" 
  alt="Example image" 
  loading="lazy" 
/>
```
