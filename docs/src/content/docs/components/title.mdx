## Props
Extends [Endomorphic](/guides/macromorphic), [Tag](/components/tag)
```ts
interface Props
  extends Endomorphic<Props, 'title', string>,
    ComponentProps<typeof Tag> {}
```

## Defaults
```js
const { as = 'h2', contrast = true, } = Astro.props
```

## Examples
### Color
```astro live
---
import { Title } from '@fulldevlabs/fullui'
---

<Title>The quick brown fox jumps over the lazy dog</Title=>   
<Title color="accent">The quick brown fox jumps over the lazy dog</Title=>  
```

### Contrast
```astro live
---
import { Title } from '@fulldevlabs/fullui'
---

<Title contrast={false}>The quick brown fox jumps over the lazy dog</Title=>   
<Title contrast={false} color="accent">The quick brown fox jumps over the lazy dog</Title=>  
```

### Compact
*It might seem weird having sizes and also compact, but it has a lot of benefits. More on this later*
```astro live
---
import { Title } from '@fulldevlabs/fullui'
---

<Title size="large" compact>The quick brown fox jumps over the lazy dog</Title>   
<Title size="medium" compact>The quick brown fox jumps over the lazy dog</Title>  
<Title size="small" compact>The quick brown fox jumps over the lazy dog</Title>   
```
