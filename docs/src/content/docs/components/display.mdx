## Props
extends [BuildProps](/guides/components), [Tag](/components/tag), [Writeup](/components/writeup), [Rating](/components/rating), [Icon](/components/icon), [Specs](/components/specs)
```ts
interface Props
  extends BuildProps<'display', Props>,
    ComponentProps<typeof Tag>,
    Pick<ComponentProps<typeof Writeup>, 'writeup' | '_writeup'>,
    Pick<ComponentProps<typeof Rating>, 'rating' | '_rating'>,
    Pick<ComponentProps<typeof Icon>, 'icon' | '_icon'>,
    Pick<ComponentProps<typeof Specs>, 'specs' | '_specs'> {
  align?: 'start' | 'center' | null
}
```

## Examples
The examples down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).

```astro live
---
import { Display } from '@fulldevlabs/fullui'
---

<Display
  tagline="This is a display",
  title="Displays",
  description="Lorem ipsum dolor sit amet, <br/> consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et."
  rating=3
/>
```
