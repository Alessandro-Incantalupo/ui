You are able to print out a predefined array of buttons with the `ButtonGroup` components.

:::note
This component is currenty in beta.
If you find any problems or bugs report them here: [GitHub Issues](https://github.com/fulldevlabs/fullui/issues).
:::

## Props
extends [Tag](/components/tag) and [Button](/components/button).
```ts
buttons: array | null | undefined
```

## Example
```astro live
---
import { ButtonGroup } from '@fulldevlabs/fullui'

const myButtons = [
  {html: "First Button"},
  {html: "Second Button"},
  {html: "Third Button"},
]
---

<ButtonGroup buttons={myButtons} />
```

You are also able to specify a `variant`, `color`, `contrast`, `size` and an `icon`.
```astro live
---
import { ButtonGroup } from '@fulldevlabs/fullui'

const myButtons = [
  {html: "First Button", size: "3", variant: "soft", icon: {name: "arrow-right"}},
  {html: "Second Button", size: "2", variant: "surface", contrast: true},
  {html: "Third Button", size: "1", variant: "ghost", color: "accent"},
]
---

<ButtonGroup buttons={myButtons} />
```
