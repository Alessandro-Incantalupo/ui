## Props
extends [Endomorphic](/guides/components), [Tag](/components/tag)
```ts
interface Props
  extends Endomorphic<'input', Props, string>,
    ComponentProps<typeof Tag> {}
```

## Example
The example down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).

```astro live
---
import { Input } from '@fulldevlabs/fullui'
import { Label } from '@fulldevlabs/fullui'
---

<Label label="Email input" />
<Input input="email" type="email" />
```
