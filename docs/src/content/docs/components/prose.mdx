## Props
extends [Endomorphic](/guides/components), [Tag](/components/tag)
```ts
interface Props
  extends Endomorphic<'prose',  Props, string | CollectionEntry<ContentCollectionKey>['render']>,
    ComponentProps<typeof Tag> {}
}
```

## Example
The example down below uses regular syntax, but this component is compatible with [all syntaxes](/guides/components#syntax).

```astro live
---
import { getEntry } from 'astro:content'
import { Prose } from '@fulldevlabs/fullui'

const collectionEntry = await getEntry('docs', 'prose')
---

<Prose prose={collectionEntry} />

```
