A container used for full-width blocks which sets:

- Outer margins and paddings
- Layout inside the container
- Spacing between child elements

```astro live
---
import { Button, Heading, Paragraph, Section } from '@fullui/ui'
---

<Section>
  <Heading>Lorem ipsum dolor</Heading>
  <Paragraph>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod.
  </Paragraph>
  <Button>Lorem</Button>
</Section>
```

## Props

### `frame`

**Type:** `'fill' | 'viewport' 'panel' = 'fill'`

If set to `viewport` the section has a minimum height of 100vh.

If set to `panel` margins are added around certain child elements and backgrounds and borderes are applied according to your [color variables](/overview/colors). You can use the variant prop explained below to adjust these styles.

```astro live
---
import { Heading, Paragraph, Section } from '@fullui/ui'
---

<Section frame="fill">
  <Heading> Lorem ipsum dolor </Heading>
  <Paragraph>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod.
  </Paragraph>
</Section>

<Section frame="panel">
  <Heading> Lorem ipsum dolor </Heading>
  <Paragraph>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod.
  </Paragraph>
</Section>
```

### `direction`

**Type:** `direction?: 'vertical' | 'horizontal' = 'vertical'`

```astro live
---
import { Button, Heading, Section } from '@fullui/ui'
---

<Section direction="vertical">
  <Heading>Lorem ipsum dolor </Heading>
  <Button>Button</Button>
</Section>
<Section direction="horizontal">
  <Heading>Lorem ipsum dolor </Heading>
  <Button>Button</Button>
</Section>
```

### `y`

**Type:** `y?: 'top' | 'center' | 'bottom' | 'spread' | 'fill' = 'top'`

Use this setting in combination with either direction `horizontal`, or frame `viewport`.

```astro live
---
import { Heading, Image, Section } from '@fullui/ui'
---

<Section direction="horizontal" x="fill" y="top">
  <Heading>Heading </Heading>
  <Image src="/example.jpg" />
</Section>

<Section direction="horizontal" x="fill" y="center">
  <Heading>Heading </Heading>
  <Image src="/example.jpg" />
</Section>

<Section direction="horizontal" x="fill" y="bottom">
  <Heading>Heading </Heading>
  <Image src="/example.jpg" />
</Section>
```
