The `logo` element is used to render a logo, usually used in the header.

## Props
```ts
interface Props {
  src?: string | null
  alt?: string | null
  href?: string
  class?: string
}

const { src, alt, href = '/' } = Astro.props
```

| Prop | Default | Purpose |
| ----------- | ----------- | ----------- |
| src | None | Directory of the image |
| alt | None | Text shown if the image does not load |
| href | / | Where the logo sends you (usually to the homepage) |
| class | None | Any custom styling classes to add to the logo |

## Example code

```js title="index.astro"
<Logo src="/favicon.svg" />
```

<Window>
  <Logo src="/favicon.svg" href="#" />
</Window>
