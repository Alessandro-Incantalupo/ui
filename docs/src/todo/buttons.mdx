You are able to print out a predefined array of buttons with the `Buttons` components.

## Props
extends [Tag](/components/tag) and [Button](/components/button).
```ts
items: array | null | undefined
```

## Example
Buttons in this component will by default transition from `solid` to `ghost`.
```astro live
---
import { Buttons } from '@fulldevlabs/fullui'

const myButtons = [
  {"button": "First Button"},
  {"button": "Second Button"},
  {"button": "Third Button"},
]
---

<Buttons items={myButtons} />
```


You are also able to specify a `variant`, `color`, `contrast`, `size` and an `icon`.
```astro live
---
import { Buttons } from '@fulldevlabs/fullui'

const myButtons = [
  {"button": "First Button", "size":"3", "variant":"soft", "icon":"arrow-right"},
  {"button": "Second Button", "size":"2", "variant":"surface", "contrast":true},
  {"button": "Third Button", "size":"1", "variant":"ghost", "color":"accent"},
]
---

<Buttons buttons={myButtons} />
```
